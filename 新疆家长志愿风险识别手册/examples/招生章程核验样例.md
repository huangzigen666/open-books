## 样例一：石河子大学 2026 章程风险卡

可写入书稿的核验点：

| 章程点 | 家长翻译 | 风险处理 |
|---|---|---|
| 分省招生计划最终以教育部审批后由省级招生主管部门公布为准 | 不要只看学校宣传页，要回到新疆招生计划专业目录 | 方案里保留招生计划来源 |
| 平行志愿投档调档比例不超过 105% | 进档不等于自动录取到想去专业 | 仍要看专业排序和调剂 |
| 专业录取不设专业志愿分数级差，按分数优先原则录取 | 专业顺序仍重要，但不会因为第几专业志愿扣级差分 | 按孩子真实偏好排序专业 |
| 所有专业志愿无法满足且服从调剂时，调剂到未完成计划专业 | 服从调剂不是调到喜欢专业，是调到未满专业 | 先列不可接受专业，再决定是否服从 |
| 英语专业只招收英语语种考生 | 非英语语种考生不能只看分数 | 外语语种必须进入信息采集卡 |
| 软件工程（NIIT班）只录取有该专业志愿考生；非英语语种慎报；入学后不允许转专业 | 合作办学/特色班不是普通专业，费用、语言、转专业都要看 | 单独列入高风险核验项 |
| 部分专业参照体检指导意见，色弱、色盲存在不能录取风险 | 体检限制不是医学类才有 | 查体检结论和专业限制 |

## 样例二：新疆师范大学 2026 章程风险卡

可写入书稿的核验点：

| 章程点 | 家长翻译 | 风险处理 |
|---|---|---|
| 分省招生计划报教育部审核后由各省招生主管部门公布 | 学校章程不是最终计划表 | 回到新疆招生计划专业目录 |
| 体检标准执行普通高校招生体检工作指导意见；身体状况不适合教师职业者慎报师范类 | 师范类也有身体和职业适配问题 | 师范志愿要看身体条件和职业适配 |
| 英语、翻译专业只招收高考外语语种为英语的考生 | 语种不符会影响报考 | 信息采集必须问外语语种 |
| 普通类专业按分数优先、遵循专业志愿，不设专业级差 | 专业顺序按真实偏好排，不是随便填 | 专业顺序需家庭确认 |
| 航空服务艺术与管理有身高、视力、色觉、体态、心理等条件 | 艺术类/特色专业限制很多，不能只看文化分和专业名 | 单独走特殊类型核验 |
| 数学与应用数学、信息与计算科学、应用统计学等专业有数学单科要求 | 总分够不代表单科够 | 单科成绩必须进表 |
| 所报专业志愿均无法满足时，服从调剂则在未完成计划专业限额内调剂；不服从则退档处理 | 调剂风险和退档风险必须二选一承担 | 调剂底线必须提前写清 |
| 新生入学后全面复查，信息不符或弄虚作假会取消入学资格 | 录取不是所有风险终点 | 材料真实性必须留痕 |

## 样例三：新疆大学 2026 章程风险卡

可写入书稿的核验点：

| 章程点 | 家长翻译 | 风险处理 |
|---|---|---|
| 学校分乌鲁木齐校区和喀什校区 | 同校不同校区会影响生活和成本 | 校区写进家庭确认 |
| 招生计划及专业以各省级招生考试管理部门公布为准 | 章程不能替代新疆招生计划目录 | 回到新疆计划目录 |
| 设计学类需省级美术与设计类统考或联考本科成绩合格 | 艺术类不是只看文化分 | 查专业统考资格 |
| 专业录取不设级差，按投档成绩优先和专业志愿顺序录取 | 专业顺序仍影响录取结果 | 按真实偏好排序 |
| 服从调剂时在未完成计划专业限额内调剂，不服从则退档 | 调剂和退档必须提前选 | 列不可接受专业 |
| 软件工程专业收费高于普通理工类专业 | 学费也是家庭决策变量 | 单独确认成本 |

## 样例四：新疆医科大学 2026 章程风险卡

可写入书稿的核验点：

| 章程点 | 家长翻译 | 风险处理 |
|---|---|---|
| 按“分数优先，遵循志愿”原则录取 | 总分和专业顺序都要看 | 专业排序要家庭确认 |
| 特殊要求专业除外，服从调剂可调到未完成计划专业 | 医学类调剂不能只看学校名 | 先列不能接受专业 |
| 法医学、护理、哈医学、维医学有志愿和调剂限制 | 有些专业不接收普通调剂 | 单独标红 |
| 色盲、色弱考生多数专业不予录取 | 医学类体检限制强 | 体检结论必须进表 |
| 口腔医学惯用左手考生慎重报考 | 细节限制可能影响长期学习就业 | 做专业适配确认 |
| 外语公共必修课为大学英语 | 非英语语种有学习风险 | 非英语语种单独提醒 |

## 样例五：昌吉学院 2026 章程风险卡

可写入书稿的核验点：

| 章程点 | 家长翻译 | 风险处理 |
|---|---|---|
| 公共外语主要为大学英语，另开大学日语、大学俄语 | 小语种考生不能默认没影响 | 查语种匹配 |
| 英语、商务英语、交通运输有外语单科要求 | 总分够不代表单科够 | 单科成绩进表 |
| 服从调剂则调剂到其他专业，不服从则退档 | 调剂不是只调到喜欢专业 | 先列不可接受专业 |
| 色弱、色盲、颜色识别异常涉及多类专业限制 | 非医学类也有体检红线 | 查体检结论 |
| 航空相关专业有身高、视力、色觉、体态等要求 | 特色专业限制集中 | 单独走特殊专业核验 |
| 4+0 应用本科只招有该专业志愿考生，培养地点在合作院校 | 培养地点和培养模式会影响体验 | 单独确认 |

## 样例六：郑州大学 2026 章程风险卡

可写入书稿的核验点：

| 章程点 | 家长翻译 | 风险处理 |
|---|---|---|
| 进档后按投档成绩和专业志愿安排专业，不设级差 | 专业顺序仍影响结果 | 按真实偏好排序 |
| 体检合格且服从调剂者，平行志愿普通批次进档后原则上不退档 | 服从调剂能降低退档风险 | 先确认调剂底线 |
| 内地新疆高中班、南疆单列、哈密和和田定向招生有单独规则 | 新疆相关计划不能套普通批逻辑 | 单独核验计划类型 |
| 色弱、色盲和颜色识别异常涉及医学、化学、管理、计算机中外合作等专业限制 | 体检限制跨专业大类 | 查体检结论 |
| 中外合作办学部分课程英语授课且收费较高 | 语种和成本都可能成为后悔点 | 单独做语种与费用确认 |

## 反面证据处理

- 新疆大学学校官网通知列表只是线索，正式引用使用阳光高考审核页，或后续补学校官网正文、微信原文 PDF、截图。
- 新疆医科大学学校官网旧页面正文显示 2025 年招生章程，不能当 2026 证据；正式引用使用阳光高考审核页。

## 通用核验维度

每所学校至少查 12 项：

1. 学校全称、办学层次、办学性质。
2. 招生计划是否以自治区公布为准。
3. 在新疆对应类别、科类、批次是否有计划。
4. 专业录取规则：分数优先、专业优先、专业级差或其他。
5. 同分排序规则。
6. 是否认可加分。
7. 体检限制或慎报说明。
8. 单科成绩要求。
9. 外语语种或外语教学限制。
10. 性别、身高、视力、色觉等特殊限制。
11. 调剂范围和不服从调剂后果。
12. 学费、校区、合作办学、定向协议等关键成本和义务。

## 不能犯的错

- 不把 2025 章程当 2026 章程。
- 不把学校官网新闻标题当完整章程。
- 不引用第三方转载内容做最终依据。
- 不把“慎重报考”当无效提醒。
- 不省略调剂后果。

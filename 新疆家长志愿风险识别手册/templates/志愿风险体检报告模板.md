# 志愿风险体检报告模板

## 一句话结论

本方案当前风险等级：红线 / 高风险 / 中风险 / 低风险

最先要处理的问题：

## 红线项

| 红线 | 证据 | 处理动作 | 是否完成 |
|---|---|---|---|
| 类别/科类未锁定 |  | 暂停方案 |  |
| 体检限报未确认 |  | 查招生章程和体检要求 |  |
| 军警公安资格未核验 |  | 查官方通知 |  |

## 高风险项

| 风险 | 为什么危险 | 建议 |
|---|---|---|
| 服从调剂后可能进入不可接受专业 | 调剂不是调到喜欢专业，是在规则范围内调 | 冲的学校必须列出组内/校内不可接受专业 |
| 志愿梯度失衡 | 全冲或全保都不是好方案 | 重新按位次和风险承受排序 |
| 数据来源不可信 | 旧分数线、截图、短视频口径可能误导 | 回到官方数据和近年位次 |

## 中低风险项

| 风险 | 处理 |
|---|---|
| 城市偏好冲突 | 家长和孩子开一次决策会 |
| 专业名称误解 | 查课程、就业、升学路径 |

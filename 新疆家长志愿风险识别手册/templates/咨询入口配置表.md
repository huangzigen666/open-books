# 咨询入口配置表

用途：正式发布前，把 Rex / 问路的真实承接入口统一配置到 `config/consultation_entry.json`。

## 必填字段

| 字段 | 当前值 | 要求 |
|---|---|---|
| status | draft / active | 没有真实入口时必须保持 draft |
| owner | 黄昊 Rex / 问路 | 统一署名 |
| service_name | 新疆高考志愿风险体检 | 对外展示的服务名称 |
| primary_action | 先做资料核验，再决定是否进入一对一方案。 | 告诉家长第一步做什么 |
| service_boundary | 只做风险识别、方案复核和决策辅助，不承诺录取结果。 | 必须保留边界 |

## 至少补一个真实入口

| 字段 | 用途 |
|---|---|
| wechat_id | 个人微信或企业微信 ID |
| public_account | 公众号名称 |
| video_account | 视频号名称 |
| booking_url | 预约链接 |
| workbench_url | 问路工作台链接 |
| qr_image | 二维码图片路径，例如 `assets/contact-qr.png` |

## 发布前检查

- [ ] `status` 已从 `draft` 改为 `active`。
- [ ] 至少有一个真实入口字段不为空。
- [ ] 二维码图片可以打开。
- [ ] 服务声明不承诺录取结果。
- [ ] 附录 C 已重新构建到 PDF。
- [ ] `assets/contact-card.svg` 已重新生成。
- [ ] `dist/contact.html` 已重新生成。

## 不允许写

- “保录取”
- “内部名额”
- “低分包上”
- “改志愿结果”
- “百分百录取”

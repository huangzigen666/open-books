# 案例脱敏处理清单

用途：把真实咨询案例处理成可以公开发布的版本。

## 基本信息

- 原始案例编号：
- 公开案例编号：
- 处理人：
- 处理日期：
- 是否取得发布许可：是 / 否 / 不需要，仅使用高度抽象信息

## 必须删除

- [ ] 姓名。
- [ ] 身份证号、准考证号、考生号、报名序号。
- [ ] 手机号、微信号、家庭住址。
- [ ] 具体中学、班级、班主任、同学姓名。
- [ ] 能反推出身份的过细地理信息。
- [ ] 截图里的二维码、头像、登录账号、浏览器账号。
- [ ] 聊天记录中的家庭隐私、疾病、冲突和非必要情绪内容。

## 可以保留

- [ ] 大致分段或位次段。
- [ ] 类别、科类、专项资格。
- [ ] 家庭诉求和决策冲突。
- [ ] 官方证据类型。
- [ ] 风险判断。
- [ ] 处理动作。
- [ ] 可复用提醒。

## 发布前检查

- [ ] 没有可识别学生身份的信息。
- [ ] 没有录取承诺。
- [ ] 没有把第三方信息写成官方依据。
- [ ] 已标注“脱敏案例”。
- [ ] 可复用点清楚，不泄露个体隐私。

# 贡献说明

这个项目是面向新疆家长的开源志愿风险识别书稿。贡献内容只接受可核验、可复核、能降低家长误判的材料。

## 可以贡献

- 官方政策链接、截图、PDF 保存说明。
- 高校招生章程中的限制项摘录。
- 脱敏案例：不能出现学生姓名、准考证号、身份证号、学校班级、联系方式等可识别信息。
- 错别字、表格、表达优化。
- HTML/PDF 发布版样式改进。

## 不接受

- 未核验分数线、位次数据。
- “保录取”“内部名额”“低分捡漏”等宣传话术。
- 可识别学生隐私。
- 把第三方平台数据当官方证据。
- 直接给具体考生推荐志愿表。

## 证据标准

1. 优先新疆教育考试院、阳光高考、招生高校官网。
2. 第三方页面只能作线索，不能作为最终依据。
3. 引用政策要写明发布日期和链接。
4. 不确定内容标注“需核验”，不要强行下结论。

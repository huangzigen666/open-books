# 案例目录

当前目录包含：

- 3 个合成案例：用于展示风险识别方法，不对应真实学生。
- 1 个脱敏真实案例：用于展示真实咨询过程中的风险告知和决策边界。

真实案例进入公开书稿前，必须满足：

- 已按 `templates/案例脱敏处理清单.md` 完成脱敏。
- 不包含姓名、证件号、手机号、学校班级、详细住址等可识别信息。
- 不公开未授权的原始咨询记录。
- 不承诺录取结果。
- 能说明一个明确的志愿风险点，而不是包装成成功案例。

建议命名：

```text
案例_主题_脱敏.md
```

示例：

```text
案例_铁路相关专业调剂风险_脱敏.md
案例_低分段技术路径选择_脱敏.md
案例_调剂专业底线冲突_脱敏.md
```

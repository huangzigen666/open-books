# 脱敏真实案例：铁路相关专业调剂风险

## 脱敏状态

- 案例类型：真实咨询过程脱敏记录
- 是否含个人身份信息：否
- 是否公开考生姓名、证件号、准考证号、手机号、毕业学校、家庭住址：否
- 是否公开具体院校名称：否
- 是否公开具体分数和位次：否
- 案例用途：呈现志愿决策中的风险识别过程，不作为录取预测样本。

## 案例表

| 项目 | 内容 |
|---|---|
| 大致分段 / 位次段 | 口述资料未提供具体分数或位次。本案例不作为分数段样本使用，只记录决策风险。 |
| 类别 / 科类 | 口述资料未说明文理科类，本案例不做科类推断。 |
| 家长原始问题 | 家庭有铁路相关行业背景，希望孩子就读铁路相关专业，倾向选择一所该方向实力较强的院校，关心能否进入目标专业。 |
| 最初误区 | 把“往年录取数据支持进校概率较高”误认为“进校后就能进入目标专业”，低估了同一院校大类内不同专业之间的调剂风险。 |
| 关键证据 | 往年录取数据只支持“被该校录取概率较高”的判断，不支持“目标专业结果已经确定”的判断；该校在相关行业方向实力较强；招生规则中同一大类下设置多个专业，存在专业调剂可能。 |
| 最终处理动作 | 规划师在决定前明确告知家庭：存在被调剂到相对冷门专业的可能；家庭与考生在知情前提下，仍共同决定将该校相关专业方向作为目标志愿。 |
| 结果边界 | 实际被该校录取，未进入最初目标专业，被调剂到同校另一专业。本案例只呈现“不确定性在决定前已充分告知”，不构成对任何院校或专业录取结果的承诺、预测或复盘模板。 |
| 后续状态 | 是否通过转专业、辅修、考研或其他路径回到原目标方向，未跟踪，无法确认。 |

## 可复用判断

这个案例的核心风险不是“学校能不能上”，而是“进入学校以后是不是目标专业”。家长容易把院校录取概率、行业认同、专业结果混成一个问题，规划师必须把三件事拆开：

- 院校层面：能否进校。
- 专业层面：能否进入目标专业。
- 后果层面：若被调剂，家庭是否能接受。

# 合成案例：专项资格误判

> 说明：这是合成案例，不对应真实学生。用途是展示专项场景的核验逻辑。

## 基本情况

- 考生：B 同学
- 方向：想报定向培养军士或公安相关方向
- 家长诉求：希望“毕业稳定、有方向”
- 当前问题：只看到了招生宣传，没核验年龄、身体、政审、体测、服役或就业义务

## 家长原始问题

“这个是不是分不高也能走？以后是不是比较稳定？”

## 判断

这个问题不能先谈分数。专项场景第一步是核资格和义务。

## 必须核验

| 项目 | 为什么重要 | 核验来源 |
|---|---|---|
| 年龄 | 定向培养军士等路径有年龄要求 | 新疆教育考试院官方通知 |
| 婚姻状态 | 部分路径要求未婚 | 官方通知 |
| 政治考核 | 不合格不能继续 | 官方流程 |
| 体检/体测 | 不合格可能无法录取或入学复查失败 | 官方通知和体检标准 |
| 服务年限 | 影响毕业后自由选择 | 招生通知/协议 |
| 入学复查 | 录取后仍可能复查 | 院校或官方通知 |

## 风险点

1. 家长只看到“稳定”，没看到绑定义务。
2. 孩子本人不了解真实训练和管理强度。
3. 资格材料不完整时，提前把普通批方案压缩得太窄。

## 处理路径

1. 先下载或打开官方通知。
2. 按条件逐项打勾，不符合就退出该路径。
3. 孩子本人写一句确认：是否接受该路径的管理和义务。
4. 同时保留普通批备选方案，不能把全部筹码押在专项上。

## 给家长的话

专项不是低分捷径，是带条件的选择。稳定不是免费的，它通常对应纪律、服务年限、地域和职业路径限制。

## 可拆内容标题

- 军警公安不是分够就能填
- 定向培养军士先看义务，再看分数
- 专项路径别只听亲戚说“稳定”

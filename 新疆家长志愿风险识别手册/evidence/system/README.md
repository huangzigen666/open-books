# 系统证据目录

本目录用于记录需要从官方系统截图或导出的材料。

公开仓库不应直接放未脱敏截图。真实截图建议先放在内部资料库，脱敏后再决定是否进入公开发布包。

## 可放入公开仓库的材料

- 已打码的成绩/位次截图。
- 已打码的志愿填报系统截图。
- 已打码的招生计划专业目录截图。
- 不含学生身份信息的截图说明文档。

## 不应放入公开仓库的材料

- 未脱敏成绩截图。
- 身份证号、准考证号、考生号、报名序号。
- 手机号、微信号、家庭住址、具体中学班级。
- 可直接识别考生身份的聊天记录。

## 建议文件命名

```text
YYYYMMDD_考生代号_证据类型_来源_版本.png
```

## 配套模板

- `templates/系统证据留存清单.md`
- `templates/案例脱敏处理清单.md`

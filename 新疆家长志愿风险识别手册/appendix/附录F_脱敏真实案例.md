## 案例一：铁路相关专业调剂风险

| 家长原始问题 | 家庭有铁路相关行业背景，希望孩子就读铁路相关专业，倾向选择一所该方向实力较强的院校，关心能否进入目标专业。 |
| 最初误区 | 把“往年录取数据支持进校概率较高”误认为“进校后目标专业结果已经确定”，低估了同一院校大类内不同专业之间的调剂风险。 |
| 关键证据 | 往年录取数据只支持“被该校录取概率较高”的判断，不支持“目标专业结果已经确定”的判断；该校在相关行业方向实力较强；招生规则中同一大类下设置多个专业，存在专业调剂可能。 |
| 最终处理动作 | 规划师在决定前明确告知家庭：存在被调剂到相对冷门专业的可能；家庭与考生在知情前提下，仍共同决定将该校相关专业方向作为目标志愿。 |
| 结果边界 | 实际被该校录取，未进入最初目标专业，被调剂到同校另一专业。本案例只呈现“不确定性在决定前已充分告知”，不构成对任何院校或专业录取结果的承诺、预测或复盘模板。 |
| 后续状态 | 是否通过转专业、辅修、考研或其他路径回到原目标方向，未跟踪，无法确认。 |

## 这个案例真正说明什么

这个案例的核心风险不是“学校能不能上”，而是“进校后是不是目标专业”。

家长容易把三件事混成一个问题：

- 院校层面：能否进校。
- 专业层面：能否进入目标专业。
- 后果层面：若被调剂，家庭是否能接受。

规划师的责任不是把不确定性讲成确定性，而是在填报前把不确定性摆到桌面上，让家庭用真实后果做选择。

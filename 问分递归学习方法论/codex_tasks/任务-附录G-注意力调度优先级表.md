## 3. 必含结构（附录＝卡片/模板页，**不套八段模板**）

按「卡片页」组织，建议如下小标题与内容（顺序可微调，内容不可缺项）。**卡的所有可填行只给表头/字段/空格 + 一行括注提示怎么填，不要替读者把某一项排好当结论**（合成示范除外，且只示范一行、清楚标成示例）。

1. **开头一句用途说明**（1–2 句自然句，并成自然段，不要一句一段堆）：这是配合第 6 章的两张通用卡；进程调度（就是 CPU 一次只能认真干一件事、不停切换会白白耗掉力气）讲的是「时间和注意力别一直在切换中空转」，落到你手上就是两件事——先修又常错又好提的、再量一量自己一坐下能专注多久；卡只给框架与空格，哪几项该排前面、你一个番茄钟被打断几次，照着填、自己判断。回链 [第6章·进程调度](../chapters/06_进程调度.md)。（CPU/进程调度首次出现各跟一句大白话。）

2. **卡一·失分项优先级排序表（空白可填，本附录主体）**：教读者用「失分频率×可提升空间」自己给本周失分项排队。先用一两句把排法说透：
   - **失分频率**＝这类题/这个点最近多久错一次、错得多不多（常错的排前面）。
   - **可提升空间**＝这一项现在补、好不好补、补了能不能很快见效（好提的排前面）。
   - **两个一起看**：又常错、又好提的，排最前面先修；很少错、或一时半会很难提的，往后放。**这是排顺序，不是把一天按小时排满。**
   给一张表，列建议覆盖：**失分项（自造脱敏，如「某类应用题最后一问」） / 失分频率（高·中·低或一周错几次） / 可提升空间（好提·一般·难提） / 这周排第几个修 / 这一项先做的那一个动作**。
   - **给一行合成示范**（自造、脱敏，明确标成「示例」），演示怎么综合「频率高 × 好提」把某一项排到第一个修、并写下先做的那个动作；示范只演示「怎么把两栏合起来排出一个顺序」这套通用做法本身，不替读者下结论。
   - **再给 3 行左右空白可填行**（只给表头与空格），让读者把自己的失分项填进去自己排。表下补一句：失分项从哪来——可以从最近的错题里挑（错题怎么归类见[第4章](../chapters/04_调试错题当bug.md)，这里不展开），挑出最常错的几类填进来排；行不够自己往下加。

3. **卡二·番茄钟中断划正字自测表（落 CPU 比喻的那个糙动作）**：把第 6 章「一个番茄钟内被自己中断几次，划一笔正字」落成可填的小表。先一两句说清怎么用：设一个番茄钟（一段你打算专心做一件事的时间，比如 25 分钟），这段时间里只要你自己中途跑去看手机、发呆、想别的、换去做另一科，就划一笔正字；一个番茄钟结束，数数划了几笔。
   - 给一张表，列建议覆盖：**这个番茄钟在做哪一件事 / 时长 / 中途被自己打断划正字（正字记） / 一共几笔 / 下一个番茄钟想改的那一个小动作（如手机放门外）**，留 3 行左右空格让读者连记几个番茄钟。
   - 表下补一句**怎么读这张表、不下个性化结论**：笔数多，说明注意力一直在切换、力气耗在切换上了，不是你不努力；先盯着把笔数往下压一点（比如把手机放远、一次只摊开一科的东西），不必追求一次到零。**点明这是给自己量一量、自己调，不是给自己打分判好坏。**

4. **给家长的一段（话术微结构·写死，对齐附录E/F 家长段）**：严格三件套，全部具体到可照说，并成自然段，**禁**「理解孩子/营造氛围/多鼓励/多沟通」类抽象词：
   - **信号**：孩子书桌上同时摊开好几科、一会儿翻这本一会儿翻那本；坐下没几分钟就摸手机、起身找东西、跟你搭话，半天没真正做进去一件事；什么都想补，结果哪科都没补扎实。
   - **一句该说的话**（示例方向）：「这一个番茄钟，咱们只做这一科这一件事，中途你想干别的就划一笔，结束咱们看看划了几笔。」
   - **一句别说的话**（示例方向）：「这么多要补的，你怎么还有空玩手机？给我全都抓紧！」
   - 补一句：家长不用替孩子排该先做哪科，只盯一个动作——孩子有没有一次只摊开一件事、把笔数往下压，而不是几科横跳、坐着空耗。

5. **卡末一句软服务邀请（全卡唯一一处，自然收口，不开边界节）**：先用一句把方法收住（这套「先修又常错又好提的、一次只做一件事、给自己数中断」的排序和自测，书里都给全了，你照着两张卡就能自己跑一遍）；再**最多一句**软性邀请，邀请口气，范式照抄附录F结尾的口气：「想要有人针对你某一科持续诊断、看准你到底总在哪散、按周陪你调节奏、用工具盯着你跑，那是问分在做的事。」**到此为止，不再加任何划界、守金库、防御式措辞。**

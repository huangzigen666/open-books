## 2. 本条目定位

- 类型：**正文章节·面向家长的特殊章**。**整章写给家长，语气是坐在家长对面跟 ta 说话；不套第1—8章那种"给学生正文＋章末给家长段"的八段模板，改按"家长视角"组织全章**（结构见第3节）。

- 它在书里的角色：**双读者落地章**。前八章讲的是孩子怎么把自己这台机器调试好；这一章告诉家长：你不用懂学科、不用替孩子做题，你的活是"把机器运行的环境维护好"——稳定供电、做好散热，别超频、别替 CPU 运算。是前言里给家长那条最短阅读路径（前言＋第9章＋各章末给家长段）的主干。
- 核心隐喻（**必须翻译成大白话、落到家长当天能做/能说的具体动作，不许停在比喻**）：
  - **孩子＝那台机器**：真正在跑学习这套循环的是孩子自己（呼应前八章）。
  - **供电（家长该做的正向事）**：把孩子学习要用的"电"供稳——情绪稳定（家里不天天为分数吵）、后勤到位（吃睡规律、环境安静）、信任不断电（相信 ta 在调试、不每天盯着质问）。供电不稳，机器随时断电重启，前面学的全丢。
  - **散热（家长该做的降焦虑事）**：机器一热就降频甚至死机；孩子一焦虑就学不进。家长要做的是给孩子散热（降焦虑），而不是自己先热起来、再把热传给孩子。
  - **超频（家长最常踩的坑一）**：给机器一味加压、加时长、加补习，逼它超过自己的处理能力——短期看着转得快，长期烧坏。对应"多报几个班、多刷几套、熬到半夜"。
  - **替 CPU 运算（家长最常踩的坑二）**：家长替孩子思考——直接讲答案、替 ta 整理错题、替 ta 决定先学哪科。CPU 被你代劳了，孩子自己那颗就一直没练起来。

## 3. 必含结构（**本章不套八段，按"家长视角"组织**；段标题用 `##`）

按下面顺序写，段落数可在此基础上合并/微调，但 ① ②（对照表）③（话术替换覆盖每一环）④（例会模板）⑤ 五块**必须齐全**：

1. **本章结论（加粗一段·结论先行，开门见山对家长说）**：第一行加粗写出——你（家长）不用懂学科、不用替孩子做题；你的活是**把孩子学习的运行环境维护好：稳定供电（情绪/后勤/信任）＋散热（降焦虑），不超频（别一味加压加时长）、不替 CPU 运算（别代替孩子思考）**。一句话核心动作：盯住"环境"，把手从"题"上拿开。

2. **先点破一个家长最容易陷进去的误区**（对应"拆穿迷思"，但写成对家长说的话）：1—2 句点破"家长越使劲、孩子提分越快"这个直觉错觉——很多时候家长越用力（加压、盯题、替做、传焦虑），孩子那台机器反而越容易断电、降频、死机。给一个可观察的具体现象（如：你一坐到旁边盯着，孩子反而更慢、更卡；你替 ta 把错题整理得整整齐齐，下次同类题 ta 还是不会）。点破后给正解：你该使劲的地方不在"题"上，在"环境"上。

3. **把系统隐喻翻译给家长：你是供电和散热，不是超频，也不是 CPU**：用大白话把上面四个隐喻（供电/散热/超频/替 CPU 运算）讲清，每个都落到"具体指家里哪件事"。讲清一个核心：真正跑学习这套循环的是孩子自己（前八章都是写给 ta 的），家长插手代劳，等于替 CPU 运算，孩子自己的处理器就一直没练起来。**这一段是隐喻落地，不要停在比喻，每个比喻后面紧跟一句"在家里具体就是……"。**

4. **家长该做／不该做对照表**（真干货一·必含）：给一张 Markdown 表，**两列对照**（"该做（供电/散热）" vs "不该做（超频/替 CPU 运算）"），至少 6—8 行，覆盖：情绪/家庭氛围、后勤（吃睡/环境）、信任与盯人尺度、面对一道不会的题、面对一次考砸、报班加量、替不替孩子整理错题/做决定、家长自己的焦虑管理。每行都要具体到家长照着能判断"我现在是在供电还是在超频"。**表格故意写得糙而能用，别追求面面俱到——可在表后用一句自然的话点出"这张表只列最常踩的，照着先把这几条做稳"，但不要写成"边界声明"。**

## 4. 红线（写死本条目硬约束）

1. **按最终模型放开方法（最高）**：把"家长该做什么、不该说什么、每一环怎么说、例会怎么开"这套**写透**，家长照着能自己做对。留给问分的只有服务（针对个人的持续诊断＋按周陪跑），**全章只用一处软性邀请带出**，不写"公开安全层到此/不泄密红线/哪些交给问分/本章不替它写/公开框架/书不交付"等防御式措辞，不单开边界声明节。**强自检（反向）**：家长照本章应能自己做对；做不到说明你藏了方法，放开它。
2. **整章写给家长·语气统一**：全章第二人称"你"指家长，不是孩子。**不要套第1—8章的学生八段模板**（不要写"给学生的练习卡""你今天挑一道题来拆"这种对孩子说的话）；本章的"动作"都是家长的动作（该做/该说/开例会）。涉及孩子做的方法时只复述并锚到对应章节，不在本章重教方法。
3. **隐喻必须落地（防生搬硬凑）**：供电/散热/超频/替 CPU 运算四个比喻，每个都必须紧跟"在家里具体就是哪件事"，落到家长当天能做或能说的动作；落不到动作的比喻一律删。**不要顺手带出二级计算机术语（电压/时钟频率/缓存一致性/中断之类）**——一律不写，只保留供电/散热/超频/替 CPU 运算这组能落动作的词，且每个第一次出现紧跟一句大白话。
4. **话术替换必须覆盖每一环**：第5块要逐环（七环节）给"别说/该说"一对，**不能只给一两句样板就带过**；每句都具体到家长可照说，禁"理解孩子/营造氛围/多鼓励/多沟通/多陪伴"类抽象词。出现孩子端术语只用大白话复述、锚到对应章节。
5. **家长不需要懂学科**：全章反复落实这条——家长维护的是运行环境，不是替孩子学；不要让家长去"陪做题/讲解知识点/判断对错"。个性化精准诊断交给孩子自己（按前八章自己跑）＋问分（一句软邀请）。
6. **不写玄学大词**：禁"成长型思维/潜能/聪明药/速成提分/玄学记忆术"等；提分是方法跑通的结果不是承诺，可点一句但不承诺名次。案例一律脱敏或合成，禁真实成绩/学校/获奖等可识别信息。
7. **不越界写规划/志愿**：本章只管"家长平时怎么维护孩子学习的环境"；若需提及规划/志愿，只一句话交棒，指向前两本（`../../新疆高中三年升学规划书/README.md`、`../../新疆家长志愿风险识别手册/README.md`），不展开。
8. **AI 降级**：如提到 AI，只让 AI 做整理/出变式题/陪练，不替孩子思考、不替家长做诊断决策（个性化诊断指向问分）。

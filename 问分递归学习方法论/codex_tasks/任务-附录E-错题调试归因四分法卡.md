## 2. 本条目定位

- 这是**附录**，不是章节：是一张可以直接打印出来当天就填的「空白卡」，不套八段模板，不写迷思、不写隐喻教学、不写给家长段。
- 它服务的是第4章已经训练过的那个动作：每道错题先归四类粗标签之一，再喂给下一轮重做。读者翻到这张卡，应当不用再读一遍理论就能直接动笔填。
- 卡的全部价值在「粗」与「空」：只给四个盒子和空格，把判断权留给填写者本人，把精准根因定位和定制纠偏留给问分。它和第一本「家长信息采集卡」同一纪律——只采集信息、给框架，不给结论。

## 3. 必含结构（附录=卡片/表格页，不套八段）

按以下顺序写，不要增段，不要把它写成讲解文：

1. **一级标题**：`# 附录 E：错题调试归因四分法卡`
2. **一句话使用说明（最多两句）**：说明「每道错题只贴一个粗标签，贴完就停，不写长反思」，并一句话回链第4章：`配套用法见 [第四章·调试错题当bug](../chapters/04_调试错题当bug.md)`。不要展开理论。
3. **四类粗标签速查表**：一张表，三列——`粗标签 | 一句话识别问句 | 一句话通用原则`，四行分别是「看错 / 不懂 / 记错 / 没时间」。**四行的问句与原则措辞须与第4章第2节那张表保持一致（可一字不差照搬，鼓励照搬以保一致）。** 通用原则里凡涉及 base case，沿用统一术语「能不查任何资料独立做对的最小单元」，不得改写定义。
4. **空白可填卡（核心）**：一张空表，列固定为 `题号 | 粗标签 | 下一轮重做时间 | 重做结果`。
   - 给 3 到 5 个空行（与第4章「挑三道、不要挑十道」一致，建议 3 行起，至多 5 行）。
   - 「粗标签」格里只放选项提示 `看错 / 不懂 / 记错 / 没时间`，「重做结果」格里只放 `对 / 仍错`，其余格全部留空。
   - 表下最多一行提示：`一道题只填一个粗标签；填完就停，不在卡上写根因分析。`
5. **卡末红线标注（必须写死，单独成行）**：`公开安全层到此·精准根因定位与定制纠偏指向问分`。在它前面可加最多一句过渡，把升级动作指向问分（如：精准根因定位、为什么你总在这一类崩、以及定制纠偏路径，不在本卡展开，指向问分）。

## 4. 红线（写死本条目硬约束，最高优先级，违反即返工）

1. **不泄密红线（最高·对齐第4章硬上限）**：本卡**只给四类粗分类 + 每类一句话识别问句 + 每类一句话通用原则 + 一张空白可填表**。
   - **严禁**写「如果是 X 且 Y 则用 Z 修法」这类二级修法分支或决策树。
   - **严禁**写「从粗标签继续反推精准根因」的示例（哪怕一个）。
   - **严禁**针对某学生 / 某科 / 某类题的精细诊断字段或评分逻辑。
   - 精准根因定位与定制纠偏一律指向问分；卡末必须出现 `公开安全层到此·精准根因定位与定制纠偏指向问分`（与第4章第7节末行一致）。
   - **强自检**：填完这张卡的读者，应当只能得到「这道题属于四类里的哪一类 + 下一轮什么时候重做 + 重做对没对」，**不应**得到「为什么我总在这里崩、该怎么定制升级」的闭环。若你的卡让读者无需任何外部诊断就能自己推出精准根因，就是写过头，收回到四类粗标签层。
2. **「只给框架不给结论」纪律（对齐第一本家长信息采集卡）**：表格只提供标签选项与空格，**不替任何学生预填结论、不给「示范填好」的样例行**（给空行即可，不要演示一行「填好的范例」，以免变相给出根因推断）。
3. **统一术语强制复用**：base case＝能不查任何资料独立做对的最小单元；不得另造定义。涉及「没时间」原则时与第4章一致——拆到 base case 提速，不用刷整套卷假装提速。
4. **不写玄学 / 不承诺提分**：禁「速成提分 / 玄学记忆术 / 押题保分」等表达；本卡是方法工具，提分是结果不是承诺。不在卡上出现名次、保过类措辞。
5. **不越界**：不写规划 / 志愿内容；本卡只管「错题怎么粗归因 + 下一轮重做」。若需指向后续产品线，只交给问分，不在卡内展开前两本内容。
6. **链接规范（防断链）**：回链第4章用 `../chapters/04_调试错题当bug.md`，锚点用可读中文文字而非把路径当锚点。不自造文件名简写。

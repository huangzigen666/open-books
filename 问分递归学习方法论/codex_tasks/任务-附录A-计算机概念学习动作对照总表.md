## 2. 本条目定位（一句话）

附录 A 是**全书速查总表 + 反术语堆砌的总锚**：一张三列对照表，把书里真正会用到的核心计算机/递归概念，逐行落成「**动词开头、当天就能做的学习动作**」和「**做完留下什么痕迹**」。它同时是一把裁剪尺——**凡填不出具体当天动作的概念，整行删除，也不准进正文**。读者用它当速查卡（忘了某概念对应啥动作就来查一行）；Codex 写各章时用它当护栏（带出的比喻在这张表落不出动作就砍）。

## 3. 必含结构（附录＝对照总表页，**不套八段章节模板**）

按这个顺序写，全文短：

1. **一句话开场（加粗，结论先行）**：点明这张表怎么用——「每个计算机概念在这里只占一行，对应一个当天能做的动作和一种留痕方式；查到哪行，今天就去做那一个动作。**填不出当天动作的概念，本书不收。**」（1–2 句，别展开成段落。）
2. **主表：一张三列总表**，表头固定为 `计算机/递归概念 | 学习动作（动词开头·当天可做） | 留痕方式`。
   - **行覆盖**（覆盖全书真正用到的核心概念，每行一个，约 11 行）：递归调用（拆解为子问题）、base case、自引用、回溯、收敛、编译、内存/缓存、进程调度、调试、版本迭代、自举/元认知。
   - **「学习动作」列硬性写法**：每格**必须动词开头**，动词从「写 / 默 / 讲 / 拆 / 标 / 勾」里选（或同类可当天执行的动词），且是**今天一次就能做完的一个具体动作**，不是「要理解…」「学会…」这种状态。示例口径（按这个颗粒度写，别照抄）：
     - base case → 「**拆**到能不查任何资料独立做对的最小一步，**写**下这一步是什么。」（统一术语：base case＝能不查任何资料独立做对的最小单元）
     - 调试 → 「**标**：给今天一道错题贴『看错/不懂/记错/没时间』四类粗标签之一。」（接第4章与附录 E，**到四类粗标签为止，不在表里展开精准根因**）
     - 收敛 → 「**讲**：把这块讲给别人听懂，或连续 N 次独立做对，到了就停。」（统一术语：收敛判据＝连续 N 次独立做对／能讲给别人听懂）
     - 内存/缓存 → 「**默**：合上书默写本节核心概念，标出默不出来的。」
   - **「留痕方式」列写法**：写**做完留下什么可复现的痕迹**（一张卡 / 一行记录 / 一段默写 / 一个标签 / 一笔正字），并尽量指到对应那张附录卡的规范名链接（见各概念对应附录：base case→附录C、编译→附录D、调试→附录E、内存缓存→附录F、进程调度→附录G、版本迭代→附录H、自举元认知→附录I）。留痕是「能被下一轮翻出来重跑的东西」，不是「写一段感想」。

## 4. 红线（写死本条目硬约束·违反即不通过）

1. **裁剪令（本附录的命门）**：**「学习动作」列每一行必须动词开头（写/默/讲/拆/标/勾 一类）、且是当天一次能做完的具体动作。凡某个计算机概念填不出这样的当天动作，整行删除——不准为了凑行写「要理解/要掌握/培养…意识」这种伪动作。** 并在表下明文写出这条裁剪规则。
2. **二级术语禁入**：**寄存器、中断向量、指针等二级术语，因落不出当天动作，不准进表、也不准进正文。** 本表就是各章砍这些词的依据，自己先不能破例。
3. **统一术语强制复用（逐字对齐，不要改写定义）**：
   - base case ＝**能不查任何资料独立做对的最小单元**；
   - 收敛判据 ＝**连续 N 次独立做对／能讲给别人听懂**。
   这两条在对应行里必须按此口径出现。
4. **不泄密红线（最高）**：本表只给「概念→当天动作→留痕」的公开框架层。**调试行只到『看错/不懂/记错/没时间』四类粗标签为止，严禁在表里写二级修法分支或精准根因；自举/元认知行只给一个当天可做的自检动作，严禁写「如何从某档升到下一档」的升级路径。** 末尾写一句边界：精准诊断/精准根因/升级路径/持续陪跑指向问分。**强自检**：若读者照这张表就能搭出「无需任何外部诊断、持续自我升级的完整闭环」，就是写过头，收回到「框架+当天动作+留痕+找问分」层。
5. **不写玄学大词**：禁「成长型思维 / 潜能 / 聪明药 / 速成提分 / 玄学记忆术」等；提分是方法跑通的结果，不是承诺。
6. **三列必须自洽**：每一行的「动作」要能产出「留痕」列那个痕迹（动作→痕迹一一对得上），不许动作和留痕各说各的。
7. **链接用规范名 + 可读中文锚点**：交叉链接一律用第六之二节规范文件名，从 `appendix/` 引用其它附录用 `附录X_….md`，锚文字用可读中文（如「附录E·错题调试归因四分法卡」），**不要把文件路径当锚点文字**。交棒前两本用 `../../新疆高中三年升学规划书/README.md`、`../../新疆家长志愿风险识别手册/README.md`。

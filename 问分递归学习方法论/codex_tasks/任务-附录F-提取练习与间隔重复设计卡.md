## 2. 本条目定位（一句话）

给读者一套**通用、可当天套用**的「提取练习设计法 + 间隔重复起步节奏框架 + 脏缓存自查动作」空白卡，把第 5 章「存进去≠取得出、重读手感顺≠记得住」这一纠偏落成可填的练习卡与自查表。**只给通用框架与起步参数方向，不做个性化诊断、不排个人专属记忆计划、不写玄学速记神技。**

## 3. 必含结构（附录＝卡片/表格页，**不套八段模板**）

按「设计卡页」组织，建议如下小标题与内容（顺序可微调，内容不可缺项）：

1. **开头一句用途说明**（1–2 句）：这是配合第 5 章的通用练习卡；自测优先于重读；卡片只给框架与空格，不替你判断该背什么、不给个性化记忆方案。回链 [第5章·内存与缓存](../chapters/05_内存与缓存.md)。
2. **卡一·提取练习设计卡（自测优先于重读）**：给**三种通用提取方式**——①合书默写（盖住材料，把核心点默写出来）②自问自答（先出题再答，对照原文）③讲给别人听（讲不顺＝没取出来）。每种配「怎么做一句话 + 一个可填空格」。**核心纠偏写死：重读手感顺 ≠ 记得住；判据＝能不看材料独立复现／讲到别人能听懂**（复用统一收敛判据措辞）。给一张**空白练习卡**（列：今天要测的核心点 / 用哪种方式（默写·自问自答·讲给别人）/ 不看材料做对了吗（对·卡住）/ 卡在哪一句），只给表头与空格。
3. **卡二·间隔重复起步节奏框架（只给起步方向，不写玄学记忆术）**：**只给起步参数的方向性区间**（如「学完当天先测一次→隔 1～2 天再测一次→隔约一周再测一次；做对了就拉长间隔，卡住了就缩短间隔」），**明确写这是通用起步节奏、不是精确遗忘曲线公式、不是个人专属计划**；给一张**空白排期卡**（列：核心点 / 首测日期 / 第二次（隔几天）/ 第三次（隔几天）/ 每次结果（对·卡住）/ 下次间隔拉长还是缩短），只给框架与空格。
4. **卡三·脏缓存自查动作**：给**当天可做的两个动作**——①定期合书默写核心概念清单②把默出来发现记错／过时的标记出来、清理掉重背一次（复用第 4 章「记错→去查它该长什么样、再重背一次」的口径，但**不展开成诊断分支**）。给一张**空白自查表**（列：核心概念 / 这次默对了吗（对·记错·想不起）/ 记错的正确版本是 / 何时重测），只给表头与空格。
5. **给家长的一段（话术微结构·写死）**：严格按「**这一环家长能观察到的信号（孩子背书/复习时的具体表现）＋ 一句该说的话 ＋ 一句别说的话**」三件套，具体到可照说。**禁「理解孩子／营造氛围／多鼓励／多沟通」类抽象词。**（示例方向：信号——反复读课本／划线很多但一合上书就讲不出；该说的话——「合上书，把这一节讲给我听听，卡在哪句我们标出来」；别说的话——「多读几遍自然就记住了／你就是不用功。」）
6. **边界标注（写死）**：结尾一句——**「公开安全层到此·这是通用提取与间隔重复起步框架；针对你某一科的个性化记忆诊断、精准节奏定制与持续纠偏指向问分。」**

## 4. 红线（写死本条目硬约束，违反即不通过）

> 把全局七条口径中与本附录相关的焊进来：

1. **不泄密红线（最高）**：本附录**只给通用练习框架 + 起步参数方向 + 自查空白卡**。**严禁**给「针对某学生某科的个性化记忆方案／精准间隔参数表（精确到第几天第几小时的遗忘曲线公式）／持续陪跑纠偏机制／一套无需外部诊断就能自动跑到收敛的完整记忆系统」。所有卡片**只给框架与空格、不替读者下结论**（对齐第一本采集卡）。**强自检：若读者照抄就能无需任何外部诊断完成「精准个性化记忆诊断 + 持续自我升级闭环」，就是写过头，收回到框架层。** 结尾必须有边界标注（见 3-6）。
2. **不写玄学记忆术 / 速记神技**：**禁**「记忆宫殿包你过目不忘／速记神技／玄学记忆术／速成提分／聪明药」等表达；提分是方法跑通的结果，不是承诺。间隔重复**只给起步方向区间**，**不得**包装成「科学精确遗忘曲线，照此必记牢」的玄学公式。
3. **三层落法**：每个概念（提取练习 / 间隔重复 / 脏缓存）必须落到**当天就能做的动作或一张可填卡**；落不到动作的比喻一律删。本附录主体就是「可填卡」，**每张卡都要真能当天填**。
4. **统一术语强制复用**：判据措辞统一为「**能不看材料独立做对／能讲给别人听懂**」（＝收敛判据口径）；提到「拆到会做的最小单元」时用 **base case＝能不查任何资料独立做对的最小单元**，并以可读中文锚点链到 [第2章·递归调用](../chapters/02_递归调用拆到会做.md)（如有引用）。
5. **双读者**：卡片正文写给考生；给家长段**写死成「信号＋该说的话＋别说的话」话术微结构**，禁抽象词。
6. **卡片纯净**：所有「空白卡」只给**表头/字段/空格**，**不要填入示范答案当结论**（可在表头下用一行括注说明怎么填，但不替读者判断他该背什么、节奏该多长）。
7. **不越界**：不写规划/志愿内容；如需交棒，仅一句话指向前两本，用相对路径 `../../新疆高中三年升学规划书/README.md`、`../../新疆家长志愿风险识别手册/README.md`（本附录通常无需交棒，能不写就不写）。

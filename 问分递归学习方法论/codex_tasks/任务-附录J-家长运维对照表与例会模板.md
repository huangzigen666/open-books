## 2. 本条目定位（附录卡·面向家长）

- 类型：**附录卡（面向家长）**，**不套八段模板**，按「卡片/模板」组织（对齐附录F 的卡体例）。
- 标题（第一行一级标题，照抄）：`# 附录 J：家长运维对照表与家庭运维例会模板`
- 它在书里的角色：**第9章「给家长的运维手册」的随手工具卡**。第9章讲透「家长是供电和散热、不是超频」的道理；这张卡把道理落成家长当天就能用的两件东西——一张覆盖每一环的该做/不该做＋话术替换对照表，一份能直接照着开的家庭例会模板。
- 卡定位（这张卡要解决家长的什么真实困境）：
  - **真实困境**：家长想帮忙，但不懂学科，一开口就是「你怎么又错」「这么简单都不会」「多刷几道」，越帮孩子越关上门；想跟孩子聊学习，要么变成查岗、要么变成吵架，没有一个稳定的、不伤人的沟通形式。
  - **这张卡给的**：①把「家长在每一环到底该做什么、不该做什么」列清楚，配上「这句别说、换成这句说」的话术替换，家长照着换嘴就行，不用懂题；②一个轻量的家庭例会框架（频率/议程/谁说什么），把「聊学习」从随机唠叨变成一周一次、有固定形式、不查岗的短会。
- 统一术语强制复用（家长看得懂的大白话，不堆术语）：
  - 把「错题/不会的题」说成**「定位 bug」**：bug 就是一处一做就崩的地方，第一次出现紧跟一句大白话（如「bug，就是一做就出错的那一处」）。家长不用懂题，只问「这道题卡在哪一步」，不评判对错。
  - 「能自己做对的最小一步」沿用全书主词（不查任何资料、自己就能独立做对的那一步），家长段只用大白话，不出现英文 base case。
  - 收敛＝**连续做对几次／能讲给别人听懂**，家长只用这一句大白话判断「是不是真会了」，不展开。
  - 全书心智模型沿用：**孩子＝那台要自己调试的机器；家长＝供电（情绪、后勤、信任）和散热（降焦虑），不是超频（不替孩子运算、不加压催逼）**。这组隐喻在卡里点一句即可，主体是对照表和例会模板。

## 3. 必含结构（附录卡＝卡片/模板组织，按下列分块写；每块用 `##` 小标题）

> 不套八段。整篇＝开头一句话说明 + 三张卡 + 家长段 + 结尾一句软邀请。表格全部给「框架＋空格」，让家长能照填照说。

1. **开头一句话说明（顶部，不加小标题或用引言块）**：一两句点明这张卡配第9章、给家长用、不用懂学科；点破判据——**家长不负责把题做对，只负责维护孩子能安心运行的环境，外加每周一次不查岗的短会**。对齐附录F 顶部那句「自测优先…照着填、照着调就行」的写法。

2. **卡一：家长该做／不该做运维对照表（覆盖每一环 + 话术替换）**：本附录的主体。
   - 按**学习系统七环节**逐环给行（环节名与附录B/第1章一致），外加「整体心态」一行收口，确保**覆盖到每一环、不是只给一两句样板**（这是第9章 brief 的硬要求）。
   - 用一张主表，列固定为：`环节（这一环在管什么·一句大白话） | 家长该做（具体动作） | 家长别做（具体踩坑） | 别说这句 → 换成这句（话术替换）`。
   - 每行的「话术替换」必须是**一句具体的、可照说的话**，左边是常见伤人旧话、右边是定位 bug 的新话；禁「多鼓励/多沟通/理解孩子/营造氛围」类抽象词。示例口径（自造、可改写，保持这个味道）：
     - 拆题环：`「这么简单都不会？多刷几道」→「先别急，这道题里哪一小步是你已经会的？从那步往下拆看看」`
     - 调试错题环：`「你怎么又错了」→「这道题的 bug 定位到哪一步了？是看错、不懂、记错，还是没时间」`
     - 记忆环：`「多读几遍自然就记住了，你就是不用功」→「先合上书讲给我听，卡在哪句我们标出来」`
     - 注意力环：`「别玩手机了赶紧学」→「这一个时间块你只做一件事，几点到几点，手机先放我这」`
     - 复盘迭代环：`「这次怎么又退步」→「这周你修好了哪一个 bug？下周想先改哪一个」`
   - 七环可参照（以附录B 实际环节为准，名称对齐）：输入/拆题（第2章）、编译·学过≠会用（第3章）、调试·错题（第4章）、记忆·内存缓存（第5章）、注意力·进程调度（第6章）、版本迭代·复盘（第7章）、元认知·自己管自己（第8章）。每环「该做/别做/话术」都要落到家长当天能做、能照说，**家长不用懂这一环的学科内容，只维护环境、只换嘴**。

3. **卡二：家庭运维例会轻量模板（频率／议程／谁说什么）**：把「聊学习」变成一周一次的固定短会。
   - 先一句话框定：**每周一次、15 分钟以内、固定时间、孩子主讲家长主要听**；它不是查岗会、不是批评会，是「看这周修好了哪个 bug、下周先改哪个」的复盘短会。
   - 给一张「议程表」框架，列固定为：`环节 | 时长 | 谁说 | 说什么（一句话引导/可照说的话）`，建议 4–5 步，例如：
     - 开场（孩子说）：这周哪一科、哪一类题让我最难受。
     - 战报（孩子说）：这周修好了哪一个 bug、是怎么修的（能讲给家长听懂＝真会了）。
     - 卡点（孩子说）：还有哪个 bug 反复卡、自己拆不动。
     - 下周一件事（孩子定、家长记）：下周只先改一个 bug，是哪个。
     - 收尾（家长说）：我这周帮你做什么后勤（接送/作息/少催一句），把这条记下来。
   - 给一张「会后留痕」空白小表，列如：`日期 | 本周修好的一个 bug | 还卡着的一个 bug | 下周只改的那一个 | 家长这周负责的后勤一件`，三四个空行让家长照填。
   - **谁说什么写死**：孩子主讲（说卡点、说战报、定下周一件事），家长只做三件——听、记、定自己的后勤；明确写「家长不评判对错、不当场纠题、不加码任务」。这把例会和查岗会区分开。

4. **给家长的一段（话术微结构·写死，对齐附录F 结尾家长段）**：严格三件套，全部可照说，禁抽象词。
   - **家长能观察到的信号**：一聊学习孩子就闭嘴或顶嘴；家长一开口就是分数、名次、「别人家孩子」；家里没有一个固定的、不吵架的聊学习的时间。
   - **一句该说的话**：例如「这周咱们开个 15 分钟的小会，你说这周修好了哪个 bug，我就听、帮你记，不查你。」
   - **一句别说的话**：例如「考这么点分还有脸开会，我看你就是不上心。」
   - 补一句：家长不用懂题，照着对照表换嘴、照着模板每周开一次短会，孩子就更愿意把卡点讲出来——讲得出卡在哪，才修得动。

5. **结尾一句软服务邀请（自然收口，不开防御节）**：
   - 先一句自然收束：这套对照表和例会模板，家长自己就能用起来，照着换嘴、照着开会就行。
   - **全附录唯一一处软邀请**（自然一句、邀请口气、不划界）：例如「想要有人针对孩子某一科一直盯着往下诊断、按周陪你们一起调，那是问分在做的事。」对齐附录F 结尾那一句的味道，不写「公开安全层到此/哪些交给问分」等字样。

## 4. 红线（写死本条目硬约束）

1. **按最终模型放开方法（最高）**：把「家长每一环该做/别做、怎么换话术、怎么开例会」这套**写透**，家长照这张卡就能自己把对照表用起来、把例会开起来。留给问分的只有服务（针对孩子的持续诊断＋按周陪跑），**全附录只用一处软性邀请带出**，不写「公开安全层到此/不泄密红线/哪些交给问分/本附录不替它写/公开框架」等防御式措辞，不单开边界声明节。**强自检（反向）**：家长照卡应能自己开起来；做不到说明你藏了东西，放开它。
2. **覆盖每一环（防只给样板）**：该做/不该做对照表必须**按七环节逐环给行**（外加整体心态一行），每环都有具体的「该做/别做/话术替换」，禁止只给一两环样板就收尾——这是第9章 brief 的硬要求。环节名与附录B/第1章一致，不自造别名。
3. **话术替换写死成可照说的话**：每条话术替换是「别说这句 → 换成这句」的**具体句子**，左旧右新，全部能照说；**禁「多鼓励/多沟通/理解孩子/营造氛围/多陪伴」类抽象词**。
4. **家长不用懂学科**：全篇口径是「家长只维护运行环境＋每周开一次不查岗短会，不替孩子做题、不当场纠题、不加码任务」；个性化诊断交给孩子自己＋问分。例会里孩子主讲、家长只听记定后勤，写死。
5. **术语只用大白话**：bug＝一做就出错的那一处（首次出现配大白话）；「能自己做对的最小一步」用大白话、不出现英文 base case；收敛＝连续做对几次/能讲给别人听懂，一句话带过。不堆二级术语，凡用一个计算机词第一次出现紧跟大白话。

7. **不越界写规划/志愿**：本附录只管「家长平时怎么维护环境、怎么开例会」；若需提及规划/志愿，只一句话交棒，指向前两本（`../../新疆高中三年升学规划书/README.md`、`../../新疆家长志愿风险识别手册/README.md`），不展开。

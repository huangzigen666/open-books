## 3. 必含结构（附录卡＝卡片/模板组织，小标题用 `##`／`###`）

> 整体体例对齐附录 F：开头一句话点用途＋链回第8章；每张卡＝小标题 + 一两句怎么用 + 表格（含「可填空格」列）；结尾家长段三件套 + 一句软邀请。

1. **开头引一句（用途 + 链回章节）**：一句话说清这卡干嘛用、配哪章。例：「这是配[第8章·自举与元认知](../chapters/08_自举与元认知.md)使用的随手卡。元认知，就是回头看自己是怎么学的、然后动手调它。卡里给的是问句框架和定档信号，具体填什么、怎么调，照着填、照着走就行。」（自举/元认知首次出现各跟一句大白话。）

2. **卡一·元认知三问随手卡**：
   - 一两句说明：每天或每周拿一个具体对象来问这三问，别空问。
   - 三问固定为（对齐目录设计第43行）：①**我现在卡在系统哪一环**（输入／编译／存储／调度／执行／调试／迭代，对应第1章七环节，一句话带过、可锚[第1章](../chapters/01_系统总览地图.md)）；②**我这周用的这个方法，这周真有效吗**（拿可验证现象判断，不是感觉）；③**下一步我自己会怎么调**（改一个变量，对齐第7章「一次只改一个变量」）。
   - **每问都配当堂可填对象**：用一张表，列「三问 / 这周拿来问的具体对象（哪个方法·哪道题） / 我的回答 / 据此我打算改的那一个动作」，留空格让读者填。
   - 给一行已填好的合成示例（自造、脱敏，别用真实学生/题目）做样板，让读者照仿；示例只演示「怎么对着一个具体对象把三问问完、落到改一个动作」这套通用做法本身。

3. **卡二·五阶段自测表**：
   - 一两句说明：照「识别信号」给自己对号入座，定完档别停，照「往下一档怎么走」看下一步。强调五档是描述你现在学习的成熟度、不是给人贴标签，档会随练习往上走。
   - 用一张表，列：**阶段 / 这一档的识别信号（你最近是不是……式，可自测、不是定义） / 到下一档的通用进阶方向（一句话、人人适用的一般做法）**。五档逐行写：
     - **模仿**：照着例题/老师步骤能做对，但换个样子就不会，说不清为什么这么做。
     - **调试**：自己错题能归类、能定位卡在哪一步（呼应第4章四分法），开始对着修。
     - **重构**：不只是修单题，开始回头改自己的学习方法本身（比如把背改成自测、把刷题改成拆题）。
     - **自我编程**：能给自己设计练习、定节奏、按计划跑，不必每步等人教。
     - **进化**：方法稳定后还会定期回看、主动升级，把「学习如何学习」变成习惯（自举收口）。
   - **每档的「进阶方向」给通用一般做法即可**（如模仿→调试：「下次做完一道会的题，逼自己讲出每一步为什么这么做，讲不出的那步就是下一个要补的」），**写透到读者照着能自己往上够一档**；这不是针对某个具体学生当前状态的自适应升级路径——那种「盯着你一个人按周往上推」是问分的陪跑，自然落在结尾那一句软邀请里，不在表里硬标注、不写「升级路径不给」。
   - 给一个「学生自己发现一个学习 bug」的合成小示例（自造、脱敏），串一下：用三问发现「这周靠重读、自测几乎没做」→ 定位自己其实卡在「调试」往「重构」这一档 → 照进阶方向改一个动作（把重读换成合书默写）。示例只演示自己跑通这一圈，不演示「持续按周盯着调到收敛」。

4. **给家长的一段（话术微结构·写死，对齐附录 F 结尾段）**：严格三件套，全部具体到可照说，禁「理解孩子/营造氛围/多鼓励/多沟通」类抽象词：
   - **这一环家长能观察到的信号**：孩子很努力但从不回头看自己「是怎么学的」；问他这周方法有没有用，只会说「还行／不知道」，说不出具体证据；一直用同一套老办法，错了也不换。
   - **一句该说的话**：例如「这周你换了什么学法没有？挑一个方法，说说它这周到底帮你做对了哪道题。」
   - **一句别说的话**：例如「别瞎琢磨怎么学了，老老实实多做题就行。」
   - 补一句：家长不用懂学科，只盯一个动作——孩子有没有偶尔停下来看自己怎么学、并动手调一个，而不是一条道闷头走到黑。

5. **结尾一句软服务邀请（全卡唯一，自然、邀请口气，不开防御节）**：对齐附录 F 结尾那句的口气。例：「这两张卡你自己就能用起来——自己问三问、自己定档、自己看下一步往哪走。真到反复卡在同一档上不去、自己调了几轮还在原地，问分会针对你这一科持续诊断、按周陪你往上调。」——这是长期盯着一个人才做得了的事，跟照卡自测是两码事。**不写任何「公开安全层到此/升级路径不给/哪些交给问分」字样。**

## 4. 红线（写死本条目硬约束）

1. **按最终模型放开方法（最高·并废止旧严格条）**：占位桩与 CODEX 第四节里「五阶段禁升档路径、三问不配怎么调」**作废**。本卡要写透：三问怎么落、五档怎么定、**每档到下一档的通用进阶方向**，读者照卡能自己跑通一轮、看清下一步。留给问分的只有「针对个人的持续诊断＋按周陪跑」，**全卡只用结尾一处软邀请带出**，不写「公开安全层到此/不泄密红线/升级路径＝陪跑内核/哪些交给问分/本卡不替它写/公开框架」等防御式措辞，不单开边界声明节。**强自检（反向）**：读者照卡若给了档却看不出怎么往上走，说明你藏了方法，放开它。
2. **三问必配当堂可填对象（防元话术空转）**：三问每一条都要绑一个具体对象（本周哪个方法／今天哪道题），并落到「改的那一个动作」；禁止停在「监控自己的学习/觉察自己的状态」这类空话。卡里必须有让读者当场填的空格列。
3. **五档每行 = 识别信号 + 进阶方向**：识别信号写成「你最近是不是……」式可自测句、不是抽象定义；进阶方向给一句人人适用的一般做法、写到能照着做。两列都不许空泛。
4. **统一术语强制复用**：触底/最小单元一律用「能自己做对的最小一步」，**不出现英文 base case**；收敛判据＝连续 N 次独立做对／能讲给别人听懂（仅一句话、可锚第7章）；自举用大白话「把学习方法本身也当成一道可拆可调的题」。不得自造别名。
5. **双读者·家长段写死**：家长段必须是『信号（具体表现）＋一句该说的话＋一句别说的话』三件套，全部可照说，禁抽象词，并补一句「家长只盯一个动作」。
6. **不写玄学大词**：禁「成长型思维/潜能/聪明药/速成提分/天赋决定/玄学记忆术」等；五档是学习成熟度的描述、不是给人贴聪明标签；提分是方法跑通的结果不是承诺。案例一律脱敏或合成，禁真实题目/成绩/学校/获奖等可识别信息。
7. **不写决策树式精细诊断**：三问与五档给的是通用框架与一般进阶方向；**不写「如果你是 X 且 Y 则……」式针对某生某科的自适应决策树**——那种贴着一个人按周调的活，落在结尾软邀请里指向问分即可，正文不编造问分内部量表/参数/评分逻辑。
8. **不越界写规划/志愿**：本卡只管「平时怎么看自己怎么学、怎么调」；若需提及规划/志愿，只一句话交棒指向前两本（`../../新疆高中三年升学规划书/README.md`、`../../新疆家长志愿风险识别手册/README.md`），不展开（一般本卡用不到，能不提就不提）。
9. **去 AI 味·顺中文语感**：表格之外的说明文字，把连续的一句一段并成自然段；少机械排比、少对仗腔、去翻译腔；像懂行的人坐你对面讲话。

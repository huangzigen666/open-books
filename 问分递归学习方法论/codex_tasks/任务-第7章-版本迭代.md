## 2. 本条目定位

- 类型：**正文章节**，套八段模板（见 CODEX 第三节）。
- 标题（第一行一级标题，照抄）：`# 第七章：版本迭代——每周给自己发一个新版本，复盘是写更新日志`
- 它在书里的角色：**递归脊柱「回溯与收敛」的承接章**。第2章定义了「能自己做对的最小一步」，第4章把错题喂进下一轮，本章把这条循环收口成一个**有节奏、能停下来的迭代**：每周发一版、回退到最近做对处带修正重跑、用收敛判据判断「这一类够了、可以停」。「收敛判据」由本章给硬定义（与第2/3/4章的引用对齐）。本章给的是人人能套的「发版本＋写日志＋一次改一个变量＋按收敛判据收口」的通用方法，写到读者照着能自己跑一轮。
- 三层落法（必须三层都落齐，缺一层即不合格）：
  - **真实困境**：每周/每次考完都「复盘」，但复盘就是对着卷子叹口气、写一行「下次要更细心/更努力」，或者把错题抄一遍感动自己；下周照旧错同一类，因为根本没记下「上次到底改了什么、有没有起作用」。另一种：一发现方法不对就推倒重来、从头再学一遍，把上几周做对的也一起扔了；或者一道题反复练到考试前一天还在练，永远不知道「练到什么程度算够」。
  - **隐喻（版本迭代/回溯与收敛）**：把自己的学习方法当成一个软件。复盘不是写感想，是**发一个新版本**：基于上一版，只改动几处，写一份更新日志记下「这版改了什么、修了哪个毛病、还剩什么没解决、下一版只试哪一件事」。做错时不是推倒重来，是**回退到最近一个你确实做对的节点**，带着这次的修正从那里重跑（这就是回溯）。一类题练到**连续 N 次独立做对／能讲给别人听懂**，就给它打个「稳定版」标记、收工，不必练到考试为止（这就是收敛）。
  - **当天动作**：现在就开一份属于你自己的更新日志（一张纸或一个文档都行），按四栏写下这一周/这一版：本版改动、修复的毛病、遗留问题、下版只试一件事；并对手头一类题用收敛判据自测一次「它到稳定版了没有」。配[附录H·每周学习更新日志模板](../appendix/附录H_每周学习更新日志模板.md)当天套用。
- 统一术语强制复用（**本章是「收敛判据」的定义源头之一，要给足大白话与硬定义**）：
  - **「收敛判据」**：本章给硬定义，可自测、不许是形容词——**连续 N 次独立做对（不查任何资料、独立做对，N 自己定，建议 3 次起），或者能把它讲给一个没学过的人听懂**，满足任一条＝这一类收敛了、可以打稳定版收工，不必刷到考试为止。首次出现给一句大白话括注（如「收敛，就是一类题练到稳了、不再忽对忽错，可以放下了」）。与第2/3/4章引用口径完全一致，不得自造别名。
  - **「能自己做对的最小一步」**：本章主词沿用第2章这句大白话（指不查任何资料、自己就能独立做对的那一步）；回退/回溯就是退回到「最近一个能自己做对的节点」，提到时一句话带过、锚第2章，不重新定义。
  - **「回退到最近做对处带修正重跑」**：本章的回溯说法，用大白话讲透——不是从头重学，是退回最近一个确实做对的地方，只带上这次的修正，从那里往下重跑。
  - **「一次只改一个变量」**：用大白话讲清——一版只动一处（换一个方法 / 改一个节奏 / 调一处习惯），下一版才看出到底是不是这一处起的作用；一次改五样，分不清是哪样有用、哪样添乱。

## 3. 必含结构（章节＝八段模板，按 CODEX 第三节逐段写；段标题用 `##`）

1. **本章结论（加粗一段，结论先行）**：第一行加粗写出——拆穿「复盘＝反思一下/感动自己」；建起「像发版本一样一周迭代一次、用更新日志记改动、用收敛判据判断该停」的能力；一句话核心动作（开一份更新日志，每周按四栏写本版改动/修复的毛病/遗留问题/下版只试一件事，并用收敛判据自测哪类可以收工；配附录H）。

2. **先拆穿一个迷思**：1-2 句点破「复盘＝反思一下/感动自己」。点出真相：对着卷子叹气、写一行「下次更细心」、把错题抄一遍，这些都不是复盘，是情绪宣泄；真复盘必须留下「这版改了什么、有没有起作用」的可追的记录，否则下周还错同一类。给可验证现象：每次都复盘、每次都写「要更努力」，可一连几周错的是同一类题；或者一不顺就推倒重来、上几周做对的也一起丢。迷思与本章方法严格咬合，不标题党。

3. **这个概念对应你学习里的哪个真实困境**（三层落法前两层）：先讲困境（复盘沦为叹气/抄题/写决心、下周照旧；或一错就推倒重来、丢掉已经做对的；或一类题练到考前都不知道够没够），再给「版本迭代」隐喻命名。讲清：你的学习方法就是一个会不断更新的软件，复盘＝发一个新版本＝基于上一版只改几处、并写清改了什么。把「我这周没复盘好/又白复盘了」重命名为「我这一版没写更新日志，也没只改一个变量，所以分不清哪步有用」。在此自然引出本章的两个承接——回退（回溯）与收敛——并一句话锚到它们各自在递归脊柱里的位置。

4. **怎么发一个版本——回退到最近做对处带修正重跑（含「一次只改一个变量」硬规矩）**：把「发版本」讲成可操作的一套动作，不许停在比喻：
   - 发版本不是从头重学，是**回退到最近一个你确实能自己做对的节点**（呼应第2章「能自己做对的最小一步」），只带上这次要修的那一处，从那里往下重跑。这样上几周的成果不会被一起扔掉。
   - **一次只改一个变量**（给硬规矩并解释为什么）：一版只动一处——换一个解题方法、或改一个学习节奏、或调一个习惯，三选一，别一次全改。下一版回看才说得清到底是不是这一处起的作用。一次改五样，下版无论变好变坏，你都不知道该归功/归咎给谁。
   - 点明边界感（自然口吻，非防御）：判断「这一版到底改好没有、偏偏你这一科为什么反复卡同一类」这种针对你个人的持续判断，是要有人一周周盯着才做得准的——这一点放到本章最后那句邀请里点一下即可，正文这里只把通用规矩讲透。

5. **每周更新日志·四栏模板（实操，与附录H对齐）**：给一张可照填的更新日志表，**四栏写死**，与附录 H 同口径：

   | 栏目 | 填什么 |
   |---|---|
   | 本版改动 | 这一版我只改了哪一处（一处，对应「一次只改一个变量」）。 |
   | 修复的毛病 | 这一版把上一版哪个反复犯的毛病修掉了/在修。 |
   | 遗留问题 | 这版没解决、留到以后的是什么。 |
   | 下版实验一项 | 下一版只试这一件事（同样只列一项）。 |

   再配一段「怎么用这张表」的示范：拿**一个合成的、脱敏的例子**（自造，别用真实学生/成绩/学校），具体走一遍——某人上一版「数学大题第二步总卡」，这一版只改一处（比如把第二步拆到能自己做对的最小一步单练，呼应第2章），更新日志四栏怎么填；过一周回看，凭更新日志判断这一处改动到底有没有让那类题变稳。示范要具体到能照着仿做，但**只演示「怎么发版本、怎么填日志、怎么凭日志判断改动有没有用」这套通用方法本身**，把方法给透。

6. **收敛判据：练到什么程度算够（硬定义 + 当天动作 + 附录H）**（三层落法第三层）：
   - 给收敛判据的**硬定义**（本章是定义源头之一，要写死，可自测）：一类题/一个知识点，**连续 N 次独立做对**（不查任何资料、独立做对，N 自己定、建议 3 次起），**或者能讲给一个没学过的人听懂**——满足任一条，就给它打「稳定版」标记、收工，不必刷到考试为止。首次给一句大白话括注。
   - 讲清它解决的真实痛点：很多人要么练几遍觉得「会了」其实没稳、要么练到考前还在练同一类、永远不敢停。收敛判据就是替你画一条「可以放下了」的线，把省下的时间挪给还没收敛的那类。
   - 显式承接递归与全书：收敛判据是递归循环的「停机条件」——没有它，复盘就成了刷到考试为止的死循环；这条判据第2/3/4章都引用过，本章给足定义。一句话锚到第4章（错题喂下一轮跑到收敛为止）即可，不展开。
   - 当天动作：今天就开一份更新日志，按四栏写下这一版；再挑手头一类题用收敛判据自测一次（连续做对几次了？能讲给别人听懂吗？），决定它该继续练还是可以打稳定版。引用[附录H·每周学习更新日志模板](../appendix/附录H_每周学习更新日志模板.md)：一张「本版改动/修复的毛病/遗留问题/下版实验一项＋收敛判据自查」的空白模板，卡只给框架与空格、不替你下结论。

7. **给家长的一段（话术微结构·写死，对齐 03 章第8段）**：严格写成三件套，全部具体到可照说，禁「理解孩子/营造氛围/多鼓励/多沟通」类抽象词：
   - **这一环家长能观察到的信号**：孩子每次考完/每周末嘴上说「复盘了」，但复盘就是对着卷子发呆、写一句「下次要更努力」，看不到改了哪一处、修了哪个毛病；同一类题连着几周还在错；或者一不顺就把整科推倒从头来，把之前做对的也一起丢。
   - **一句该说的话**：例如「这周你只改了哪一处？写下来——改了什么、修掉了哪个毛病、下周只试哪一件事，下周我们对着看它有没有用。」
   - **一句别说的话**：例如「天天说复盘，怎么还错同一道？复盘有什么用，多做几套不就好了。」
   - 补一句：家长不用懂这门学科，只盯一个动作——孩子有没有把「这周只改了一处、改了什么」写下来留痕，而不是空喊更努力或推倒重来。

   - 先用一两句自然收束：这套「一周发一版、写更新日志、一次只改一个变量、按收敛判据收工」的方法，书里都给透了，你一个人就能照着一版一版往下迭代。
   - **全章唯一一处软服务邀请**（自然一句，邀请口气，不划界）：例如「方法都在书里了，你自己就能一周周发版本；真到自己迭代两三轮还总卡在同一类毛病、判断不准这一版到底改好没有，问分会盯着你这一科一直往下挖、按周陪你迭代到收敛。」——这是长期陪着一个人才做得了的事，跟看书自学是两码事。
   - **本章自查**：用 `- [ ]` 复选框清单（沿用 03 章格式），至少 6 条，覆盖：这周的复盘有没有写成更新日志（四栏齐）、而不是叹气/抄题/写决心；本版是不是只改了一个变量；做错时有没有回退到最近做对处带修正重跑、而不是推倒重来；有没有用收敛判据（连续 N 次独立做对／能讲给别人听懂）判断哪类可以打稳定版收工；有没有用附录H 模板；有没有真开一份日志、当天填了一版。

## 4. 红线（写死本条目硬约束）

1. **按最终模型放开方法（最高）**：本章把「发版本、写更新日志、一次只改一个变量、回退到最近做对处重跑、收敛判据怎么硬判」这套通用方法**写透**，读者照着能自己跑一轮迭代、跑到收敛。留给问分的只有服务（针对个人的持续诊断＋按周陪跑），**全章只用一处软性邀请带出**，不写「公开安全层到此/不泄密红线/哪些交给问分/本章不替它写/公开框架/按周持续纠偏机制指向问分」等防御式措辞，不单开边界声明节。**强自检（反向）**：读者照本章应能自己发版本、迭代到收敛；做不到说明你藏了方法，放开它。
2. **三层落法（防生搬硬凑）**：版本迭代/发版本/更新日志/回退重跑/收敛判据都必须①绑真实困境（复盘沦为叹气抄题、推倒重来、不知练到几遍算够）②给隐喻③落当天动作或可填卡。落不到动作的比喻一律删。**不要顺手带出 git/commit/分支/回滚/语义化版本号/CI/灰度发布这类二级术语**——在附录A 填不出当天动作就一律砍，只保留「发版本/更新日志/回退到最近做对处重跑/一次只改一个变量/收敛判据」这组能落动作的核心词；凡用一个计算机词，第一次出现紧跟一句大白话解释。
3. **统一术语强制复用**：收敛判据＝连续 N 次独立做对／能讲给别人听懂（本章给硬定义、首次大白话括注，口径与第2/3/4章完全一致）；「能自己做对的最小一步」沿用第2章口径、不重新定义，回退即退回到最近一个能自己做对的节点。术语不得自造别名。
4. **双读者·家长段写死**：第7段家长段必须是『信号（具体表现）＋一句该说的话＋一句别说的话』三件套，全部可照说，禁抽象词。
5. **结论先行加粗 + 迷思严格对应**：第一行加粗结论；开场只拆「复盘＝反思一下/感动自己」这一个迷思，且与「发版本/写更新日志/收敛判据」方法严格咬合，不标题党。
6. **不写玄学大词**：禁「成长型思维/潜能/聪明药/速成提分/玄学记忆术」等；提分是方法跑通的结果不是承诺，可在适当处点一句但不承诺名次。案例一律脱敏或合成，禁真实题目/成绩/学校/获奖等可识别信息。
7. **不越界写规划/志愿**：本章只管「平时怎么把学习方法一周周迭代到稳」；若需提及规划/志愿，只一句话交棒，指向前两本（`../../新疆高中三年升学规划书/README.md`、`../../新疆家长志愿风险识别手册/README.md`），不展开。
8. **AI 降级**：如提到 AI，只让 AI 做整理/出变式题/陪练复现/帮你把更新日志整理成表，不替考生做复盘决策、不替考生判断该不该收敛（个性化诊断指向问分）。
9. **去 AI 味·顺中文语感**：把连续的一句一段并成自然段；少机械排比，长短句搭配、有自然停顿；去翻译腔、对仗腔；像懂行的人坐你对面讲话，不像生成的文档。

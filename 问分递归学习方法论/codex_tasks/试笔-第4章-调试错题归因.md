## 2. 本章定位（一句话）

把错题从"罚抄/订正一遍"升级为"定位这次思维在哪一步崩了的 bug"。显式承接递归的**自引用**机制——错题不是用来改的，是用来**喂下一轮**的（用上一轮的错当下一轮的输入）。按 bug 类型分类，不按题型分类。**但本章只到"四类粗标签"为止，精准根因和定制纠偏是问分付费产品的活。**

## 3. 章节内统一结构（按 CODEX 第三节，8 段）

1. **本章结论**（加粗一段，结论先行）：拆穿"订正＝抄一遍"，把错题变成可定位的 bug；一句话核心动作（每道错题先归到四类标签之一，再喂下一轮）。
2. **先拆穿一个迷思**：点破"错了就改对答案/订正＝抄一遍"，给一个可验证现象（如"同一类题反复错，错题本抄得很厚，分还是不动"）。
3. **这个概念对应你学习里哪个真实困境**：先讲困境（错题反复错却说不清错在哪），再给"调试/bug 定位"隐喻命名——错题是程序崩溃的报错，不是用来罚站的。
4. **一个可上手的递归练习**：给"错题归因四分法"——**只给「看错 / 不懂 / 记错 / 没时间」四类标签，每类配一句话识别问句 + 一句话通用原则**（如：记错→去查它该长什么样、再重背一次；没时间→把这类题拆到 base case 提速，见第2章）。再显式落"喂下一轮"：归类后这道题怎么进下一轮（如隔几天不看答案重做）。引用[附录E·错题调试归因四分法卡](../appendix/附录E_错题调试归因四分法卡.md)（卡只给空白框架/标签，不给结论）。
5. **给家长的一段**（话术微结构·写死）：严格按「**这一环家长能观察到的信号（孩子做题/对错题时的具体表现）＋ 一句该说的话 ＋ 一句别说的话**」三件套写，具体到可照说。**禁止"理解孩子/营造氛围/多鼓励/多沟通"类抽象词。**（例：别说的话——"你怎么这么粗心又看错"；该说的话——"这道错题归到哪一类了？"）
6. **不泄密红线**：见第 4 节硬护栏；章末显式标注。
7. **避免玄学硬要求**：本章每个比喻必须落到一个真实学习动作，落不到就删。
8. **本章自查**：`- [ ]` 复选框清单（沿用前两本格式）。

## 4. 第 4 章硬护栏（最高优先级·违反即不通过）

这是全书泄密上限最严的一章。**严格执行 CODEX 第四节「第4章硬上限」：**

- **只给四类粗标签 + 每类一句话识别问句 + 每类一句话通用原则。** 到此为止。
- **严禁出现"如果是 X 且 Y 则用 Z 修法"的二级修法分支**；严禁完整反推根因示例；严禁针对某学生某科的精细诊断决策树。

- **章末显式标注一句：「公开安全层到此·精准根因定位与定制纠偏指向问分」。**
- **强自检红线**：写完回读——若学生照本章就能无需任何外部诊断、自己完成"精准根因定位 + 定制纠偏路径"，就是写过头了，必须收回到"四类粗标签 + 通用原则 + 找问分"层。

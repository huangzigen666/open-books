## 2. 本条目定位

- 类型：**正文章节**，套八段模板（见 CODEX 第三节）。
- 标题（第一行一级标题，照抄）：`# 第三章：编译——学过 ≠ 会用，知识要先翻译成考场能执行的指令`
- 它在书里的角色：MVP 波次三大痛点钩子之一，直击「一看就会一做就废」。本章不讲个性化诊断，只给一个公开的、人人能套的「自己当编译器、自检编译是否通过」的方法框架。
- 三层落法（必须三层都落齐，缺一层即不合格）：
  - **真实困境**：上课全听懂、例题看一遍就觉得会了，一到自己独立做、做变式、考场上就废——投入了「听」的时间，却没产出「能执行的能力」。
  - **隐喻**：源码 vs 可执行文件。老师讲的、笔记上抄的、答案看懂的，都是**源代码**；考场上你大脑能直接调用、不查资料跑出来的，才是**可执行文件**。「听懂」只是拿到了源码，没编译；「会用」是编译通过、生成了可执行文件。把「一看就会一做就废」重命名为「编译没通过/编译报错」，而不是笼统说「你要再理解理解」。
  - **当天动作**：合上书/答案，对一个今天刚学的知识点，自己当一次编译器跑三道编译判据（能复述 / 能讲给别人听懂 / 能独立做出一道变式），凡有一条卡住，按报错对照表定位是哪一类报错，当天就标出来。
- 统一术语强制复用：
  - 「不懂」类报错要退回 **base case ＝能不查任何资料独立做对的最小单元**（一句话锚到第2章，不展开）。
  - 「编译通过」的硬判据要和全书 **收敛判据＝连续 N 次独立做对／能讲给别人听懂** 口径一致：本章把「能讲给别人听懂」「能独立做出变式」作为编译是否通过的硬判据，明确这是收敛判据在「编译环」的落地。

## 3. 必含结构（章节＝八段模板，按 CODEX 第三节逐段写）

按以下八段顺序写，段标题层级对齐 `04_调试错题当bug.md`（用 `##` 一级小标题分段，可在段内再分点）：

1. **本章结论（加粗一段，结论先行）**：第一行用加粗写出——本章拆穿「听懂了/看会了就等于会了」这个迷思；建起「自己当编译器、用硬判据验收知识是否真的可执行」这个能力；一句话核心动作（合上书，对今天学的一个点跑三道编译判据：能复述/能讲懂/能做变式，卡住即定位报错）。

2. **先拆穿一个迷思**：1-2 句点破「听懂了就等于会了」。给一个可验证的具体现象（不讲空话）：例如「老师一讲就点头、例题一看就懂，合上书自己写却卡在第二步；或换个数字、换个问法就不会了」。迷思必须与本章方法严格对应，不标题党。

3. **这个概念对应你学习里的哪个真实困境**（三层落法前两层）：先讲困境（听懂≠会用、一看就会一做就废、投入了听课时间没产出可执行能力），再给「源码 vs 可执行文件」隐喻命名。讲清：听课/看答案＝拿到源代码；考场能不查资料调用＝可执行文件；中间缺的那一步叫「编译」。明确把「一看就会一做就废」当成「编译没通过」这个具体报错，而不是含糊地说「要理解透」。

4. **编译通过的硬判据（清单/对照表）**：给一张「编译通过」硬判据清单，至少含三条、每条都是可自测的硬动作，不许是形容词：
   - 能复述：合上书，能把这个知识点/解法的关键步骤完整说一遍。
   - 能讲懂：能讲给一个没学过的人听懂（对齐收敛判据「能讲给别人听懂」）。
   - 能做变式：换数字、换条件、换问法的同类题，能独立做对一道（对齐「连续 N 次独立做对」）。
   - 明确：三条全过才算编译通过；只过「合书觉得自己懂」不算，「重看一遍手感顺」不算。

5. **编译报错对照表（把「一看就会一做就废」当报错来定位）**：给一张报错对照表，把「废」拆成几类**粗报错**，每类只给「一句话识别问句 + 一句话通用方向」，对齐第4章四分法那张表的粒度和「故意很粗」的自陈。建议四类粗报错（动词/现象开头，不要堆术语）：

   | 报错现象 | 一句话识别问句 | 一句话通用方向 |
   |---|---|---|
   | 看着会，合上书复述不出来 | 我能不能合上书把关键步骤说全？ | 这是没编译，回去把这一步当源码重读再合书自测，别停在「看懂」。 |
   | 能复述，但讲不给别人听懂 | 我能不能让没学过的人听懂为什么这么做？ | 讲不通的那一步就是没编译的那行，标出来单独攻。 |
   | 原题会，换个数字/问法就废 | 同类变式我能不能独立做对一道？ | 这是只背了这一道的可执行文件，没编译成通用函数；找两道变式重跑。 |
   | 某一步根本讲不出为什么 | 这一步我能不能讲出它为什么成立？ | 这是「不懂」类，退回第2章的 base case，标出不会的最小单元再往上拼。 |

   表后写一句「这张表故意很粗，粗才安全」类的自陈（对齐 04 章），点明：它只帮你把「废」分到哪一类报错为止，不替你判断「你这一科到底为什么总在这里编译失败」——那是精准根因，指向问分。

6. **一个可上手的递归练习（三层落法第三层·当天动作）**：给一套当天就能做的动作 + 一张可填小卡（卡片只给框架/自查，不给个性化结论，对齐采集卡母本）。建议：
   - 今天挑一个刚学的知识点或一道刚听懂的例题（只挑一个，不要挑十个）。
   - 合上书/答案，依次跑三道编译判据（复述/讲懂/变式），每条记「过 / 卡」。
   - 凡「卡」，对照报错表归到一类粗报错，写下下一次重编译时间。
   - 给一张可填卡（Markdown 表格），列：`知识点/题号 | 能复述(过/卡) | 能讲懂(过/卡) | 能做变式(过/卡) | 报错类别 | 下次重编译时间`。
   - 明确：卡片只采集「编译是否通过、卡在哪类报错」，不替学生判断「你为什么会这样」。
   - 显式承接递归：把没编译通过的点喂进下一轮（呼应第4章自引用），一句话带过即可，不展开。

7. **给家长的一段（话术微结构·写死，对齐 04 章第6段）**：严格写成三件套，全部具体到可照说，禁「理解孩子/营造氛围/多鼓励/多沟通」类抽象词：
   - **这一环家长能观察到的信号**：孩子说「这个我会了/听懂了」，但一让他不看书讲一遍就卡壳；作业里原题会做、换个数字就空着；复习时一直在重看课本和答案、很少合上书自己测。
   - **一句该说的话**：例如「这个你先别看书，合上给我讲一遍，讲得通就算真会了。」（口径＝用「能讲懂」当编译判据）
   - **一句别说的话**：例如「你不是说听懂了吗？怎么还不会，那就是没认真听。」
   - 补一句：家长不用懂这道题的学科内容，只盯一个动作——孩子有没有合上书自测，而不是反复重看。

8. **不泄密红线 + 本章自查**：
   - **不泄密红线段**：写明本章只给公开框架（编译通过硬判据 + 报错粗分类 + 每类一句话方向 + 为什么）。明确本章**不**提供：针对某个学生某一科「为什么总在这里编译失败」的精准根因、定制化的「如何把这一档编译能力升到下一档」的升级路径、持续陪跑。学生能用本章做到的，是「自己当编译器、把废归到哪类报错、安排重编译」；学生不能只靠本章做到的，是无需外部诊断的精准根因定位与长期纠偏。
   - 章末单独一行显式标注（照 04 章格式）：`公开安全层到此·精准编译根因定位与定制升级路径指向问分`
   - **本章自查**：用 `- [ ]` 复选框清单（沿用 04 章格式），至少 6 条，覆盖：是否合上书跑了三道编译判据、是否停在粗报错层没把自己逼进复杂诊断、是否给卡住的点写了下次重编译时间、是否把「重看手感顺」改成了「合书自测」、是否把讲不出为什么的点退回 base case（第2章）、是否守住边界（精准根因与升级路径指向问分）。

引用附录：正文至少一处引用 `附录D`，用规范相对路径与可读中文锚文本：`[附录D·编译通过判据与报错对照表](../appendix/附录D_编译通过判据与报错对照表.md)`。说明该附录提供空白判据清单与报错对照表，卡片只给框架不替学生下结论。

## 4. 红线（写死本条目硬约束）

1. **不泄密（最高）**：本章只给「公开方法框架（编译硬判据＋报错粗分类）＋自学起步（当天自测卡）＋为什么（编码≠可执行的原理）」。**严禁**写：针对某学生某科的精准编译根因诊断、「如果是 X 且 Y 则用 Z 修法」式二级分支决策树、「如何从这一档编译能力升到下一档」的升级路径、持续陪跑机制。报错对照表**只到四类粗报错＋一句话识别＋一句话通用方向为止**，不得展开成精细诊断树。章末必须有 `公开安全层到此·精准编译根因定位与定制升级路径指向问分` 标注。**强自检**：若读者照抄本章就能在「无任何外部诊断」下完成精准根因定位＋持续自我升级闭环，就是写过头，收回到框架层。
2. **三层落法（防生搬硬凑）**：源码/可执行/编译三个概念都必须①绑真实困境（一看就会一做就废）②给隐喻③落当天动作或可填卡。落不到动作的比喻一律删。**不要顺手带出寄存器/链接器/字节码/中间码这类二级术语**——若它在附录A填不出当天动作就一律砍，只保留「源码/可执行/编译/编译报错」这组能落动作的核心词。
3. **统一术语强制复用**：base case ＝能不查任何资料独立做对的最小单元（「不懂」类报错退回它，锚第2章）；收敛判据＝连续 N 次独立做对／能讲给别人听懂（「编译通过」硬判据复用它）。术语口径不得自造别名。

7. **不越界写规划/志愿**：本章只管「平时怎么把知识编译成可执行」；若需提及规划/志愿，只一句话交棒，指向前两本（`../../新疆高中三年升学规划书/README.md`、`../../新疆家长志愿风险识别手册/README.md`），不展开。
8. **AI 降级**：如提到 AI，只让 AI 做整理/出变式题/陪练复现，不替考生跑编译自测、不替考生做诊断决策（个性化诊断指向问分）。

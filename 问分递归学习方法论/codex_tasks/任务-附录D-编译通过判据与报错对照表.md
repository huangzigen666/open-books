## 2. 本条目定位（一句话）

附录 D 是第 3 章「编译」的随手速查页：上半部分给一份**可自测的「编译通过」硬判据勾选清单**（让读者一条条勾，判断「学过」有没有真翻译成「考场能执行的指令」）；下半部分给一张**「一看就会一做就废」报错对照表**，把这种模糊体验拆成具体「报错现象」，给「报错现象 → 可能卡在哪 → 当天动作」三列通用方向——**只给通用方向，不做针对某学生某科的个性化诊断**。

## 3. 必含结构（附录＝卡片/表格页，不套八段）

按以下顺序写，全部短句、清单、表格，不写大段铺陈：

1. **一行用途说明**（仿采集卡首行）：一句话说明这张卡配合第 3 章用、怎么用——「学过一个知识点后，对照下面判据逐条勾；勾不满＝还没编译通过，别急着进下一题」。

2. **① 「编译通过」硬判据清单**（可勾选 `- [ ]`）：把第 3 章三条硬判据落成可自测勾选项，每条都是「合上书/资料后能独立做到」的动作，不是感觉。至少含：
   - [ ] 合上书和笔记，能把这个知识点/方法**用自己的话复述**一遍；
   - [ ] 能**讲给一个没学过的人听懂**，对方能照着做对；
   - [ ] 给一道**改了数字/换了情境的变式题**，能**不查任何资料独立做对**；
   - 可再加 1–2 条同口径自测项（如：能说出这个方法「什么时候用、什么时候不用」；能指出自己上一次在这一步崩在哪）。
   - 清单下写一句硬判据收口：**勾不满＝处于「源码看懂了、可执行文件还没生成」状态，按第 3 章重新编译，别当成已经会了。** 「独立做对变式」一项显式复用统一术语＝ base case 口径（能不查任何资料独立做对的最小单元）。

3. **② 「一看就会一做就废」报错对照表**（三列 Markdown 表）：表头固定为 `报错现象 | 可能卡在哪（通用方向） | 当天动作`。把模糊的「一做就废」拆成 4–6 行**具体可识别的报错现象**，每行给**通用方向**（不点名具体学科/具体学生）＋**一个当天能做的动作**。示例覆盖面（措辞自拟，保持粗与通用）：
   - 看例题秒懂、自己动笔就卡在第一步 → 可能：没把题拆到 base case → 当天动作：照第 2 章/附录 C 把这道题拆到能独立做对的最小单元；
   - 跟着答案能写、合上答案就断 → 可能：在「读懂答案」停住、没生成可执行步骤 → 当天动作：合书复述步骤、再做一道变式（对照①判据）；
   - 平时会、考场上想不起来 → 可能：提取失败（取得出是另一件事）→ 当天动作：用自测代替重读，详见第 5 章/附录 F；
   - 步骤都对、最后数字/符号老错 → 可能：偏「看错」类执行噪声 → 当天动作：按第 4 章/附录 E 归一个粗标签再重做；
   - 会做但每次都来不及做完 → 可能：偏「没时间」，熟练度不够 → 当天动作：把这类题拆到 base case 提速（第 2 章），别用刷整套卷假装提速。
   - 表内「可能卡在哪」一律只给**一个粗方向**，跨章动作用规范名链接到对应章/附录。

4. **一句边界收口 + 公开安全层标注**：表后写一句——**这张表只帮你把「一做就废」定位到一个粗方向并启动当天动作；为什么你这一科总卡在同一处、怎么从这一档精准升上去，属于个性化诊断，本书不展开。** 末行显式标注：**「公开安全层到此·个性化精准诊断与升级路径指向问分」。**

## 4. 红线（写死本条目硬约束，违反即不通过）

把全局口径中与本条相关的几条焊进这里：

1. **不泄密红线（最高）**：报错对照表**每行只给「一个粗方向 + 一个当天通用动作」**，到此为止。**严禁出现「如果是 X 且 Y 则用 Z 修法」的二级修法分支；严禁针对某学生、某科、某类题的精细诊断决策树；严禁写「如何从这一档升到下一档」的升级路径**（升级路径＝问分陪跑内核）。**强自检**：写完回读，若读者照这张表就能「无需任何外部诊断、自己完成精准根因定位＋持续自我升级闭环」，就是写过头，收回到「粗方向＋当天动作＋找问分」层。末行必须有公开安全层标注。
2. **判据是可自测的勾选项**：① 清单每条都用 `- [ ]`，且每条都是「合书后能独立做到的动作」，不能写成「我感觉差不多懂了」这类无法勾选的感觉项。
3. **不写玄学**：禁「速成提分／聪明药／玄学记忆术／成长型思维／潜能」等空话；提分是方法跑通的结果，不是承诺。不堆形容词——删掉形容词后这页必须仍是一份能勾选、能照做的工具表。
4. **三层落法/不生搬硬凑**：报错表每一行的「当天动作」必须是当天就能上手的真实学习动作，落不到动作的行删掉；不带寄存器/中断向量/指针这类落不到动作的二级术语。
5. **统一术语强制复用**：凡涉及「独立做对最小单元」一律用 base case 口径（能不查任何资料独立做对的最小单元）；不要自造同义新词。
6. **不越界**：不在本附录展开第 4 章精准根因、第 5 章记忆术细节、规划/志愿内容——需要时一句话用规范名链接到对应章/附录，不展开。
7. **案例脱敏**：报错现象用通用学习场景描述，禁出现可识别学生信息（真实成绩/学校/获奖）。

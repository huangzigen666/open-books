# 附录 H：每周学习更新日志模板

这是配合[第7章·版本迭代](../chapters/07_版本迭代.md)使用的周更日志卡。复盘不是写感想，是给自己发一个新版本：这周改了什么、修掉了哪个反复犯的毛病、还剩什么没解决、下周只试哪一项。照着每周填一张就行。

## 卡一：每周更新日志模板

复盘＝写感想、感动自己没用。判据是这一版到底改了什么、能不能验证。

| 版本号/周次 | 日期 | 本版改动（只记动作，不写心情） | 修复的毛病（反复犯的错，英文叫 bug） | 遗留问题 | 下版实验一项（只写一项） |
|---|---|---|---|---|---|
|  |  |  |  |  |  |
|  |  |  |  |  |  |
|  |  |  |  |  |  |

每周只填一行。下版实验一项必须只写一件事，别把一整张愿望清单塞进去。

## 卡二：一次只改一个变量

一次改三样，下周有效了也不知道是哪一样起作用；一版只动一处，才看得出这一改到底有没有用。下版只挑「下版实验一项」那件事去改，其它先按兵不动，一周后回看那个毛病有没有少犯。

| 这周只改的那一件事 | 改之前那个毛病多久犯一次 | 改之后多久犯一次 | 这一改算有效吗 |
|---|---|---|---|
|  |  |  |  |
|  |  |  |  |

## 卡三：收敛判据自查

收敛，就是一个学法反复跑几轮后越来越稳，不再忽对忽错。硬判据写死：连续 N 次独立做对／能讲给别人听懂＝这一项已经稳了，可以停，不必刷到考试为止。没收敛的遗留问题，带进下一版继续重跑。

- [ ] 这一项是不是连续 N 次不查资料独立做对了？
- [ ] 我是不是能把这一步讲给别人听懂，对方也能照着做？
- [ ] 这个毛病是不是连着几周都没再犯？
- [ ] 前三条满足后，我有没有把它从日志里结掉，换下一个卡点上来？

## 给家长的一段

信号：孩子周末复盘只写「这周不够努力、下周加油」，从不记自己具体改了哪一处学法、哪个错还在反复犯；同一类题一周周还在错，复盘本上全是情绪话。

一句该说的话：「别写『要加油』，就写一句：这周你在学法上具体改了哪一处，下周只打算再改哪一件？」

一句别说的话：「复盘有什么用，赶紧多做两套卷子去。」

这张周更日志和收敛判据，书里都给透了，你自己就能每周发一版、自己看出什么时候该停、什么时候该换卡点。

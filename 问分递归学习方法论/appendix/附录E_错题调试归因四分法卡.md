# 附录 E：错题调试归因四分法卡

这张卡只做一件小事：每道错题先贴一个粗标签，贴完就停，不写长反思。先从最近一次作业或试卷里挑三道错题，记下题号、粗标签、下一轮重做时间和结果就行，先别急着上升到学科判断、训练安排或自我评价。卡片可以打印，也可以抄到错题本首页，当天就能用。配套用法见[第四章·调试错题当bug](../chapters/04_调试错题当bug.md)。

## 四类粗标签速查表

先把错题归类，再照「通用修法」走一遍。这四步谁都能照着做。

| 粗标签 | 一句话识别问句 | 通用修法（人人可照做） |
|---|---|---|
| 看错 | 我是不是把题干、符号、条件、单位、范围看岔了？ | 回到原题，把看岔的地方圈出来，重新完整读一遍，再确认题目到底要你求什么。 |
| 不懂 | 我能不能合上答案，把这一步为什么这样做讲出来？ | 退回到这一类里你能自己做对的最小一步（指不查任何资料、不看答案就能独立做对的那一步），把卡住的地方单独拎出来先弄懂，再往上接。 |
| 记错 | 这个公式、概念、步骤，和课本或笔记里的原样一致吗？ | 翻回课本或笔记，看它原本长什么样，当场重背一遍，别凭印象硬写；隔天再默一次，确认是真记住了，不是临时记。 |
| 没时间 | 这题我会做，但考试时间里没来得及做完吗？ | 这类题你其实会，缺的是熟练。把它拆成你能一口气做对的最小一步，反复练那一步，把卡顿的环节练顺，别靠刷整套卷假装提速。 |

## 自己能跑的一轮诊断

光贴标签还不够。下面这套流程，不用任何人帮忙，你自己就能跑一遍：先把这道题归哪一类，再找出到底缺在哪，照那一类的通用修法去补，过几天盖住答案重做一次，看修没修上。一句话：**归类 → 找缺口 → 对着修 → 隔几天重做验证。**

这里头「找缺口」最值钱，多数人也恰恰卡在这一步。几个例子帮你对照着找：

- 如果归到**看错**，缺口往往是某个固定动作没做——比如从来不圈条件、不看单位，读到一半就开始算。把这个动作补上，下次照做就行。
- 如果归到**不懂**，缺口是某一步的道理没通。一直往回退，退到你不查资料也能独立讲清楚的那一步为止，缺口就在那一步再往上一格。
- 如果归到**记错**，缺口是你记的东西和原样对不上。把你记的和课本摆在一起比一比，差在哪一眼就看出来。
- 如果归到**没时间**，缺口通常是某个环节太慢，一步拖垮了整道题。找出最慢的那一步，单独练它。

针对修完别急着算赢。隔三五天，盖住答案重做这道题：能独立做对，这一轮才算真有结果；要还是错，就把这套流程再跑一遍，缺口未必和上次在同一处。

## 空白可填卡（带诊断流程）

每道题填一行，把上面那套流程落到纸面上。一道题只贴一个粗标签。

| 题号 | 归哪一类 | 缺口在哪（一句话） | 打算怎么修 | 下一轮重做时间 | 重做结果 |
|---|---|---|---|---|---|
|  | 看错 / 不懂 / 记错 / 没时间 |  |  |  | 对 / 仍错 |
|  | 看错 / 不懂 / 记错 / 没时间 |  |  |  | 对 / 仍错 |
|  | 看错 / 不懂 / 记错 / 没时间 |  |  |  | 对 / 仍错 |

「缺口」和「怎么修」各写一句就够，别写成长篇反思，也别再加别的字段。能把这五项稳定填下来、并且真去重做，这张卡就开始起作用了。

## 这张卡到哪儿为止

通用的归类方法、找缺口的思路、四类的通用修法，再加上这套「归类 → 找缺口 → 对着修 → 隔几天重做验证」的自查流程，书里都给全了。你照着就能自己跑一轮，把错题从「抄过答案」推到「粗归因 + 针对性重做」。

# 附录 I：元认知三问与五阶段自测卡

这是配合[第8章·自举与元认知](../chapters/08_自举与元认知.md)使用的随手卡。元认知，就是回头看自己是怎么学的，再动手调它；自举，就是把你平时用的学习方法本身，也当成一道可以拆、可以调试、可以发新版本的题。每天或每周拿一个具体对象来填，别空问「我学习状态怎么样」。

## 卡一：元认知三问随手卡

三问每次只盯一个对象：这周一个方法、今天一道题、最近一类错。问完以后，只改一个动作，接上[第7章](../chapters/07_版本迭代.md)的「一次只改一个变量」。

| 三问 | 这周拿来问的具体对象（哪个方法·哪道题） | 我的回答 | 据此我打算改的那一个动作 |
|---|---|---|---|
| 我现在在系统的哪一环？（输入 / 编译 / 存储 / 调度 / 执行 / 调试 / 迭代，先按[第1章](../chapters/01_系统总览地图.md)定位） |  |  |  |
| 这个方法这周到底有没有用？（别看感觉，看它有没有让我更能独立做对） |  |  |  |
| 下一步我自己会怎么调？写成一句「下周我把 X 改成 Y」。 |  |  |  |
| 示例：这个背诵方法这周到底有没有用？ | 生物概念复习：每天重读笔记。 | 读得顺，合书讲不出，说明卡在「存了取不出」。 | 下周把每天重读改成合书默写三条，再对照课本补漏。 |

调到什么程度算稳，别凭手感。用收敛判据：连续 N 次独立做对／能讲给别人听懂。这个判据在[第7章](../chapters/07_版本迭代.md)里已经讲过，本卡只拿来验结果。

## 卡二：五阶段自测表

五档描述的是你现在这套学习方法成熟到哪一步，不是给人贴标签。照信号对号入座，定完档就看下一步往哪走。

| 阶段 | 这一档的识别信号（你最近是不是……） | 到下一档的通用进阶方向 |
|---|---|---|
| 模仿 | 你最近是不是照着例题和老师步骤能做，题一换样子就断，还说不清为什么这么做？ | 开始给错题贴[第4章](../chapters/04_调试错题当bug.md)四类标签；讲不清的一步，按[第2章](../chapters/02_递归调用拆到会做.md)拆到能自己做对的最小一步。 |
| 调试 | 你最近是不是已经会把错题分成看错、不懂、记错、没时间，也能说出卡在哪一步？ | 把同一类反复错的题放在一起，不再只补单题，而是重写这一类题的做法，一版只改一个变量。 |
| 重构 | 你最近是不是开始改自己的学法，比如把重读改成自测，把刷整套改成先拆题？ | 每周固定跑一次三问，只改一个动作，并把改动写进更新日志。 |
| 自我编程 | 你最近是不是能自己设计练习、安排节奏、回看效果，不必每一步等人教？ | 把已经跑稳的方法迁移到新学科或新题型，用收敛判据确认它是否站得住。 |
| 进化 | 你最近是不是遇到新问题时，会自动问「我现在卡在哪一环、这办法有没有用、下版改什么」？ | 保持这套循环，定期清掉没用的方法，把有效做法留下来。 |

## 合成小例子

有个学生复习生物一直靠重读笔记。他觉得「我读得很顺」，一合书却讲不出三个关键词之间的关系。照三问走：第一问，卡在存储和提取，东西像是存进去了，考试时取不出来；第二问，重读这周没让他更能独立说出来，方法无效；第三问，下周把「读三遍」改成「合书默写三条关系，再翻书核对」。按五阶段看，他已经不只是模仿，能发现问题，处在「调试」往「重构」走的位置。下一步不是再读更多遍，而是把复习法本身改掉一个动作。

## 给家长的一段

这一环家长能观察到的信号：孩子很努力，却从不回头看自己是怎么学的；你问这周方法有没有用，他只会说「还行」「不知道」，说不出哪道题因此做对了；一套老办法用了很久，错了也不换。

一句该说的话：「这周你换了什么学法没有？挑一个方法，说说它这周到底帮你做对了哪道题。」

一句别说的话：「别瞎琢磨怎么学了，老老实实多做题就行。」

家长不用懂学科，只盯一个动作：孩子有没有偶尔停下来看自己怎么学，并动手调一个动作，而不是一条道闷头走到黑。

这两张卡你自己就能用起来：自己问三问、自己定档、自己看下一步往哪走。真到反复卡在同一档上不去、自己调了几轮还在原地，问分会针对你这一科持续诊断，按周陪你往上调。

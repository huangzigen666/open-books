# 第四章：调试——错题不是用来罚抄的，是用来定位 bug 的

**本章结论：错题不是用来罚抄的，是用来找出这次思路在哪一步崩了的。核心动作就一句话：每道错题先归到「看错 / 不懂 / 记错 / 没时间」四类之一，再针对那一类去修，过几天盖住答案重做一遍，看修没修好。**

先拆穿一个迷思：订正，不等于把正确答案抄一遍。

很多孩子的错题本又厚又花，步骤抄得工工整整，可分数就是不动。最常见的情形是：同一类题这周错、下周还错；这次对着答案看会了，隔几天盖住答案又卡住；你问他到底错在哪，他只会说「粗心」「没想起来」「当时脑子一下蒙了」。这其实算不上订正完成——他只是把出错的地方抄了一遍，却没搞清楚错在哪、为什么错、下次怎么不错。

我们这本书把学习当成一台你能自己调试的机器。「调试」是个借来的词——程序员写的程序出了毛病，他们不会瞎改，而是先弄清楚是哪一步出了岔子，再对着那一步去修。错题，就是你这台机器给你的报错信息。报错不是拿来罚站的，它是在告诉你：问题出在这儿。

## 1. 错题真正暴露的不是答案，是卡在哪一步

一道题做错，表面看是结果错了；可更要紧的，是你的脑子在某一步卡住了。卡在哪儿，各有各的样：可能是读题时把条件看岔了，可能是某个概念你压根没真正弄懂、只是背了个壳，可能是公式记成了另一个样子，也可能是其实会做、但考试时间不够没做完。

这四种错，扣分的时候长得一模一样，可处理它们的办法，绝不能糊成一句「下次认真点」。看错了，得回去把题重读一遍；真没弄懂，就得退回去补那个概念；记岔了，那就翻课本对一对原样；至于会做但来不及的，只能靠把那几步练熟、把速度提上来。四种错，四副药，抓错药等于白吃。

## 2. 错题归因四分法：先贴标签，再对症修

每道错题，先贴一个粗标签，而且只贴一个。别一上来就写长篇反思，也别急着改训练计划。第一步，是把这个错从一团糊里捞出来，看清它属于哪一类。

下面这张表，每一类都给了三样东西：怎么认出它、认出后照着改的通用办法、以及一句要守住的原则。

| 粗标签 | 怎么认出它 | 认出后照着改（通用修法步骤） |
|---|---|---|
| 看错 | 我是不是把题干、符号、条件、单位、范围看岔了？ | ① 回到原题，把看岔的那个字、那个符号圈出来；② 完整重读一遍题，确认条件没漏；③ 想一句下次提醒自己的话，比如「看到『至多』先停一秒」；④ 过几天拿同类题验证，看还会不会再看岔。 |
| 不懂 | 我能不能合上答案，把这一步为什么这么做，自己讲一遍？ | ① 找出讲不出来的那一步，就是缺口所在；② 退回到你能自己做对的最小一步——就是不查任何资料、自己就能独立做对的那一步；③ 从那一步往上，一步一步把中间补回来，每补一步都要能自己讲出为什么；④ 过几天盖住答案重讲，讲得顺了才算补上。 |
| 记错 | 这个公式、概念、步骤，跟课本或笔记里的原样，对得上吗？ | ① 翻回课本或笔记，查它到底长什么样；② 把记错的地方和正确的并排放，找出差在哪个字、哪个符号；③ 重背一遍，背完默写出来核对，别凭印象硬写；④ 过几天再默一次，确认这回记牢了。 |
| 没时间 | 这题我会做，可考试时间里没来得及做完吗？ | ① 把这类题里你最慢的那一步找出来；② 退回到你能自己做对的最小一步，单独把这一步反复练熟、练快；③ 给这类题掐表，记下自己实际花多久；④ 过几天再掐一次表，看时间有没有压下来。别靠刷整套卷子假装在提速。 |

这张表很粗，先归一类就好。它只帮你把错题放进四个盒子，再给你一条对着这一类去修的路。它不会替你判断「你这个人为什么偏偏老在这一类上栽跟头」，那是更深一层的事，后面会说清楚归谁管。

## 3. 把错题喂回下一轮

这套学习方法有个关键动作，叫「自引用」——说白了，就是这一轮的结果，会变成下一轮的材料。上一回做错的题，不是终点，是下一回的入口。

错题就是最典型的这种材料。很多人订正完就合上本子，觉得这道题翻篇了；可真正有用的做法，是把它放进下一轮，再跑一遍。

当天，只做三件事：

1. 写下题号。
2. 贴上四类标签里的一个。
3. 写下下一次重做的时间。

到了重做那天，先盖住答案，自己从头做。能独立做对，说明你这一轮真的修好了。要还是错，就再贴一次标签——标签可能没变，也可能换了一类，比如上回是「记错」，这回发现其实是「不懂」。那也正常，接着对着新标签修就是。

错题不是废纸，是下一轮的入口。

## 4. 自己跑一遍：一道错题完整的调试流程

光说四类标签还不够。这一节给你一条能自己从头跑到尾的流程，四步：**归类 → 找缺口 → 对着修 → 隔几天重做验证。** 拿一道你最近做错的题，照着走一遍。

**第一步，归类。** 别先骂自己「太粗心」。先拿四个问句过一遍：

- 题干条件有没有看岔？
- 这一步为什么这么做，我讲得出来吗？
- 用到的公式、概念、步骤，有没有记错？
- 考试时间里，是不是来不及做完？

哪个问句一问，你心里「咯噔」一下——那一类，多半就是它。只贴一个标签，先别贪多。

**第二步，找缺口。** 标签定了，就顺着上一节那张表，把缺口具体定位到点上。

- 归到「看错」，就翻回原题，把看岔的那个字、那个符号圈出来。缺口就是那一个点，不是「我整个人不认真」。
- 归到「不懂」，就找出你讲不出来的是哪一步。一直往回退，退到你能自己做对的最小一步为止——就是不查资料、自己就能独立做对的那一步。讲不出的那一段，就是缺口。
- 归到「记错」，就把你写的和课本上的并排放，圈出差在哪个字、哪个符号。
- 归到「没时间」，就把这类题里最拖时间的那一步找出来。

**第三步，对着修。** 缺口找准了，就照第二节表格里那一类的修法走，别越界去修别的。看错就练「读完题先停一秒确认条件」；不懂就从能自己做对的那一步往上、一步一步把中间补回来，每补一步都要能讲出为什么；记错就重背加默写核对；没时间就把最慢那一步单独练熟、掐表提速。一次只修一类，别想着一道题把四种病全治了。

**第四步，隔几天重做验证。** 这一步最容易被跳过，可它才是判断修没修好的唯一办法。当天就记下重做时间——隔三五天比较稳，太近了你还记着答案，验不出真假。到点了，盖住答案，从头独立做一遍：

- 做对了，这道题这一轮的调试就算收工，标个「已修好」。
- 还是错，别灰心，回到第一步重新归类。有时你会发现标签换了——上回当成「记错」，这回才看出根上是「不懂」。那就对着新标签再修一轮。

这条流程你完全可以自己跑，不需要任何人盯着。一道题、一处缺口，对着修，再隔几天复查一次——这套跑顺了，自我调试的基本功就握在你手里了。

## 5. 这件事当天怎么做

今天，从最近一次作业或试卷里挑三道错题。一道一道来，同样一次只盯一道（原因见第二章）。

每道题，照着前面那条流程走，填这四项：

| 题号 | 粗标签 | 下一轮重做时间 | 重做结果 |
|---|---|---|---|
|  | 看错 / 不懂 / 记错 / 没时间 |  | 对 / 仍错 |
|  | 看错 / 不懂 / 记错 / 没时间 |  | 对 / 仍错 |
|  | 看错 / 不懂 / 记错 / 没时间 |  | 对 / 仍错 |

填完就停。这一步看着小，可正因为它小，才扛得住天天做。

错题本之所以失败，多半不是因为孩子不会写漂亮步骤，而是每道错题都被当成一次情绪事件来处理——今天懊恼一阵，明天就忘了，下次考试又在同一个坑里栽下去。

把它压成这四项，不是图省事，是为了留下一条能复查的痕迹：哪道题、哪类毛病、哪天重跑、跑出什么结果。没有这四项，错题本再厚也很难真正进入下一轮。

你不必在当天就把自己所有的问题都想明白。当天能做到的，是把这道题从「我又错了」，变成「它现在有一个标签，也有一个下次再见的时间」。别写五百字反思，也别把一道今天的错题，扩成一整套改造计划。本章要练的，就是这么一个小动作：看见错题，先定位是哪一类，再对着那一类修，然后安排一次重做来验证。把这个动作稳稳做起来，错题本才开始真正有用。

## 6. 给家长的一段

这一环，家长能看到的信号是：孩子错题本抄得整整齐齐，同类题却反复错；一问错在哪，只会说「粗心」「没注意」「下次会了」；对错题明显烦躁，只想赶紧把订正写完交差。

该说的一句话：「这道题先别急着解释，先归一类——看错、不懂、记错，还是没时间？归完了，咱们对着那一类改，过几天再盖住答案做一遍看看。」

别说的一句话：「你怎么又这么粗心，错题抄十遍。」

家长不用去判断学科上的深层原因。你只盯一个动作：孩子有没有给错题贴上标签、对着那一类修、并安排一次重做来验证。能稳定做到这三步，方向就对了。

## 7. 这一章给到哪、剩下的归谁

把话说明白：通用的方法和能自己跑的流程，这本书都给你了。照着本章，你一个人就能完成这些——

- 把错题归到四类标签之一。
- 找到每一类的缺口在哪。
- 对着这一类用通用办法去修。
- 隔几天盖住答案重做，验证修没修好。

这一整套「归类 → 找缺口 → 对着修 → 隔几天重做验证」，你一个人就能跑通，跑通了就能解决掉大半错题。

剩下的两件事，不是看一本书就能替代的，那是问分在做的：一是针对你这一个人的持续诊断——同样是「不懂」，为什么偏偏是你、偏偏在这一类题反反复复栽，你的薄弱环节具体卡在哪条链子上，这得持续看着你才诊得准；二是按周盯着你、动态调整的长期陪跑——这周练什么、下周节奏要不要变、哪个毛病压下去又冒出来，这得有人一周一周跟着你调。

除了这两件，别的都在书里了。通用方法和自查流程，照着做就行；针对你个人的持续诊断和按周陪跑，才是问分的事。

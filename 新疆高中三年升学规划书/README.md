# 新疆高中三年升学规划书

暂定书名：**别等出分才开始：新疆高中家庭三年升学规划手册（初升高—高三 · 3+1+2）**

作者：黄昊 Rex / 问路

定位：面向新疆高中家庭的开源「三年升学规划节奏书」。它不承诺名校结果，不替代一人一策的最终方案；它只解决一个更前置的问题——在高中三年的每个学期窗口里，别错过那些过期就补不回来的提前布局，别在选科、资格、身体条件、户籍学籍年限、特殊招生路线上犯不可逆的错，最后把孩子带着干净的资格、对的选科、齐全的材料，安全交棒给第一本《别把孩子的分数浪费在志愿表里》。

这是「问路」业务型开源书系的第二本，与第一本构成同一条产品线：**第二本在前（三年长跑 · 主动规划），第一本在后（志愿季冲刺 · 被动风险识别）。**

## 与第一本的关系

| 维度 | 第二本（本书） | 第一本《别把孩子的分数浪费在志愿表里》 |
|---|---|---|
| 时间段 | 初升高暑假 → 高三上 / 出分前 | 出分后 → 填报这几天 |
| 视角 | 主动提前布局：错过如何发生 | 被动风险识别：错误如何发生 |
| 解决 | 别错过窗口、别犯不可逆错 | 别填错、必须核验什么 |
| 结尾 | 交棒章：打包三年成果给第一本 | 合格志愿方案长什么样 |

两本严格不重叠，且**重叠区一律单向交棒第一本**：本书绝不重写冲稳保排序、招生章程逐条核验、投档退档、填报前 72 小时；也不重写第一本第 4 章（招生章程体检限报核验）、第 8 章（军警/公安/医学/公费师范/南疆专项/预科/定向资格核验）、第 7 章（低分段含复读决策）。本书在身体条件、特殊路线、复读等话题上**只做「提前自查能不能走、值不值得走、来不来得及」，凡进入逐条核验章程/政审达标/体测标准/出分后该不该复读，一律一句话指向第一本对应章，不在本书展开。** 两本共用同一套写作规则与同一条业务闭环。

## 为什么选这个题

第一本把获客窗口卡在「高三志愿季那几天」。本书把它前移到「高一甚至初升高」，拉长用户生命周期，并天然向第一本导流：读者高一读本书、高三读第一本、出分后转一对一。

- 免费价值：让高一家庭立刻看懂三年节奏，避免最早、最便宜、最不可逆的错（尤其选科与户籍学籍年限）。
- 商业边界：只给通用时间轴、判断框架、自查清单、空白模板，不替任何一个孩子拍板选哪科、走哪条路。
- 产品协同：时间轴与早筛结构可反哺「问路工作台」的三年规划档案、路线可行性评估、交棒报告模块。
- 内容复用：每章、每张卡片都能拆成图文号、小红书、私域长文、课程脚本。

## 冠军架构与嫁接

主线采用评审冠军【时间轴】：把高中三年拆成七个时间段，每段只回答「这学期家庭必须完成什么」和「错过/做错会留下什么不可逆代价」。在此基础上嫁接四条贯穿副线，弥补时间轴「主动性不足、选科信任峰值被稀释」的短板：

1. **不可逆决策里程碑副线**（取自「决策里程碑」）：高中三年只有 5 件做错回不了头的事（选科定科、方向倒推、路线分流、资格早筛、院校梯度初定）。第 1 章用一张「里程碑地图」把分散在各学期的最高信任内容先聚成锋利钩子；后续每个学期章标注「当期命中哪个里程碑」。其中「路线分流」内部的**竞赛、强基、综合评价三者时间窗差异极大且都不可逆**（竞赛与综评高一就要启动持续积累，强基高三集中报名但依赖高一高二学科积累），第 1 章里程碑地图把这三者各自的关键时间窗单列标注（均标需复核），避免家长误以为「路线分流」是高二一次性决策。
2. **错过链条副线**（取自「错过链条」）：用「被动是怎么一步步攒出来的」损失链做前言/第 1 章的情绪入口，并区分「可补救的遗憾 vs 不可逆的错过」（对应第一本「结果不完美 vs 事故」），与第一本方法论血缘显性对齐。
3. **身体条件、资格资质与户籍学籍年限早筛主线**（取自「家庭能力系统」+「错过链条」）：独立成章并贯穿全书，越早越值钱、一票否决项前置，配独立早筛卡。其中**户籍/学籍年限类资格单列为独立一票否决早筛项**（专项计划常要求户籍满 3 年 + 学籍连续 3 年，南疆专项/国家专项尤其敏感，过期补不回，年限要求需复核），避免被各学期摊薄。
4. **两条横切能力层**（取自「家庭能力系统」）：信息系统（官方/需复核/小道消息三级台账）与亲子沟通分工（家庭升学例会 + 父母分工表）——官方依赖最轻、第一本完全没有，显著拉开差异并降低整书官方维护负担。

「路径分流自测卡 + 家庭信息采集卡」降级为附录级领资钩子，**输出边界写死：只输出「你大概要重点关注哪 1-2 个里程碑 + 此处建议人工复核」，严禁输出具体路线名、具体时间节点、具体准备动作清单**，判定结果分支不得超过「方向粗标 + 转人工」两段，坚决不把六/九条路径各自的完整三年规划或节奏免费倒出，守住不替代一对一的红线。

## 写作规则

对齐第一本 5 条：

1. 结论先行：每章开头先告诉家长「这章防什么错、错过什么」。
2. 不写玄学：不用「押名校」「保上线」「玄学规划」这类表达。
3. 风险/错过分层：先红线（不可逆），再高风险，再中低；并区分「可补救的遗憾 vs 不可逆的错过」。
4. 官方优先：政策、时间、选科要求、批次、体检限报、户籍学籍年限、特殊招生资格，以新疆教育考试院及当年官方发布为准，需复核处必须标「需复核」。
5. AI 降级：AI 只做材料整理、查漏、生成解释稿，不替家庭做选科/路线/复读的最终拍板。

本书特有 3 条：

6. 时间轴定位：每章先告诉家长「你家现在站在哪一段、还来得及做什么」。
7. 不可逆点显式：每章标注「本学期命中哪个里程碑、过期补不回的是什么」；并把「长期高压导致的亲子关系/孩子意愿崩塌」作为一条贯穿提醒（比选错科更难修复），在前言与第 1 章先点破，不只放在第 12 章。
8. 交棒衔接：凡涉及志愿表排序、冲稳保、章程核验、72 小时、出分后该不该复读、章程/政审/体测达标标准，一律指向第一本，不在本书展开。

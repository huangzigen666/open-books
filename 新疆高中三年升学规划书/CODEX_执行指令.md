# CODEX 执行指令（第二本 · 三年规划书）

本文件是给 Codex（执行/写正文）的规范。架构与边界已由副主编（Claude）锁定，Codex 只负责按 brief 产出正文，不改架构、不改边界。

## 一、角色与产出边界

- 用户=主编；Claude=副主编/执行主编（架构+指令）；**Codex=执行（写正文）**。
- 本仓库已成型第一本《别把孩子的分数浪费在志愿表里》。写本书前，先 Read 第一本 `chapters/01_志愿填报的第一性原理.md` 与 `chapters/00_前言.md`，并 Read 第一本 `templates/家长信息采集卡.md`（第 13 章/附录 L 要对接其字段），严格对齐其语气、句式、清单/表格密度。
- Codex 产出：`chapters/*.md`、`appendix/*.md`、`templates/*.md`，全部中文。

## 二、写作总规则

对齐第一本 5 条：

1. **结论先行**：每章第一行用加粗写出「本章结论」——这章防什么错、错过什么。
2. **不写玄学**：禁用「押名校」「保上线」「玄学规划」「内部渠道」等表达。
3. **风险/错过分层**：先红线（不可逆）→ 高风险 → 中低；并区分「可补救的遗憾 vs 不可逆的错过」。
4. **官方优先**：政策、时间、选科要求、批次、体检限报、户籍学籍年限、特殊招生资格，以新疆教育考试院及当年官方发布为准；会随年份变的一律标「需复核（以当年官方发布为准）」。
5. **AI 降级**：AI 只做材料整理、查漏、生成解释稿，不替家庭拍板选科/路线/复读。

本书特有 3 条：

6. **时间轴定位**：每章开头点明「你家现在站在哪一段、还来得及做什么」。
7. **不可逆点显式**：每章用一个小节标注「本学期命中哪个里程碑、过期补不回的是什么」（5 个里程碑：选科定科、方向倒推、路线分流、资格早筛、院校梯度初定）。其中「路线分流」涉及竞赛/强基/综评时，须分别点明三者时间窗不同（竞赛与综评高一启动、强基高三报名但靠前期积累）。并把「长期高压导致亲子关系/孩子意愿崩塌」作为贯穿提醒，在前言与第 1 章先点破。
8. **交棒衔接**：凡涉及志愿表排序、冲稳保、招生章程逐条核验、投档退档、填报前 72 小时、出分后该不该复读、章程/政审/体测逐条达标标准，一律一句话指向第一本「详见《别把孩子的分数浪费在志愿表里》第 X 章」，不在本书展开。

## 三、章节内统一模板

每章按以下结构写（标题层级与第一本一致，正文短句、多清单、多卡片/表格）：

1. **本章结论**（加粗一段）：结论先行，防什么错 + 错过什么。
2. **你家现在站在哪一段**：时间轴定位，1-2 句。
3. **这一年/这一学期家庭该做什么**：1-3 件必做事项。每件不可逆事项用「里程碑四件套」微结构写：**时间窗 + 前置准备 + 一次家庭决策 + 一份留痕**。
4. **不可逆点提醒**：本章命中哪个里程碑、过了哪个信号就补不回；用「可补救的遗憾 vs 不可逆的错过」二分法分级。
5. **家长可操作清单**：空白可填框架 / 自查问题（不给个性化结论）。
6. **需复核标注**：凡政策/时间/选科要求/资格门槛/户籍学籍年限，行内标「（需复核：以当年官方发布为准）」。
7. **何时必须人工复核**：明确指出本章哪些判断必须找专业人士，并指向咨询入口。
8. **本章自查**：复选框清单（沿用第一本 `- [ ]` 格式）。

涉及特殊路线/低分段处，统一用四段式：**适配条件 + 触发信号 + 标准动作 + 何时必须人工复核**。

## 四、与第一本的防重写硬约束（所有涉及重叠话题的章节必读）

本书与第一本在三处话题必然交集，区隔靠「时态」：**本书=提前自查（能不能走/值不值得走/来不来得及）；第一本=当下逐条核验（能不能报/怎么填/达标标准）。** Codex 在以下章节必须执行：

- 第 7 章（身体条件/资格早筛）：只写「提前自查是否可能触线、年限够不够」。凡进入「逐条核验招生章程体检限报、政审、体测达标标准」一律一句话指向第一本第 4 章、第 8 章，不写核验细则。
- 第 8 章（特殊路线分流）：只写「要不要走、来不来得及、提前准备什么」。凡进入「逐条核验某路线资格/章程/政审达标」一律指向第一本第 8 章。
- 第 11 章（PlanB·复读转轨）：只写「何时该启动评估的触发条件 + 提前留后路的准备项（如保留学籍、了解复读政策窗口）」。凡涉及「出分后该不该复读、复读分数预期」一律指向第一本第 7 章。

## 七、禁止事项

1. **不替代一对一**：不给「你家孩子应选哪三科、走哪条路、报哪所学校」的最终结论；所有路径/选科产出停在「框架+自查清单+需复核」。**强自检红线**：若某章已能让家庭照抄出一份无需任何人工复核的完整三年方案，就是写过头了，必须收回到框架层（逐章用此条质检）。
2. **附录 K/templates 路径分流自测卡输出边界写死**：判定结果分支不得超过「方向粗标 + 转人工」两段，**只输出「你大概要重点关注哪 1-2 个里程碑 + 此处建议人工复核」，严禁输出具体路线名、具体时间节点、具体准备动作清单。**
3. **附录 E 分阶段约束**：MVP 阶段只写极简版（九条路线名 + 官方查询入口 + 一句话适配方向 + 此处建议人工复核），**不写「提前布局年限」「触发信号」等可直接套用细节**；完整版随第 8 章在第二阶段发布。
4. **不写玄学**：不承诺名校、不用押中/保过类表达。
5. **不编造官方数据**：不杜撰具体分数线、位次、选科要求、资格门槛、户籍学籍年限、时间节点；不确定一律标「需复核（以当年官方发布为准）」或留空白框架。
6. **不泄露产品壁垒**：不写问路工作台的字段结构、评分算法、复核规则；只给家长能用的简版表格。
7. **案例必须脱敏或合成**：禁止可识别学生/家庭信息（真实成绩、学校、获奖、户籍轨迹）。
8. **不越界写第一本内容**：冲稳保、章程逐条核验、72 小时、出分后复读决策、政审/体测达标标准一律只做指引，指向第一本。

# 附录 D：身体条件与资格早筛清单（含户籍学籍年限）

本附录结论：**身体条件和户籍学籍年限是「越早查越值钱」的一票否决项——它们大多在出分前就已经定了，出分后才发现触线或年限不够，已无法回头补救。本附录只帮你提前自查「是否可能触线、年限够不够」，不替你做任何达标判断；逐条核验体检限报、政审、体测的达标标准，一律见第一本《别把孩子的分数浪费在志愿表里》第 4、第 8 章。**

使用方式：每一项给一个「自查问题」+「越早查越值钱的理由」。你只需在每项后回答「确定没问题 / 可能触线 / 不知道」三选一。凡是「可能触线」或「不知道」，就标红、进入人工复核，不要自己拍板「能走 / 不能走」。

判断口径写在前面：本附录只解决「这条路对我家会不会一开始就被身体或年限挡死」，不解决「具体差多少、怎么补、达不达标」。后者属于出分后逐条核验，见第一本。

---

## 1. 体检受限专业（提前自查是否可能触线）

体检限报不是只发生在医学类。色觉、视力、身高、既往病史这几类，常在你完全没预料的专业上设限。以下为大类举例，不穷举。

| 自查项 | 自查问题（确定没问题 / 可能触线 / 不知道） | 越早查越值钱的理由 |
|---|---|---|
| 色觉（色盲 / 色弱） | 孩子是否做过正规色觉检查？是否可能色弱或色盲？ | 色觉异常涉及医学、化学、生物、部分工科、设计等跨大类专业，且无法矫正，早知道才能在选科和方向上提前避开 |
| 视力 | 孩子裸眼 / 矫正视力大概在什么水平？是否可能影响军警、飞行、航海等方向？ | 视力可干预的窗口在初中、高一最宽，拖到高三才发现已来不及 |
| 身高 / 体态 | 孩子身高是否可能卡在军警、空乘、航海类的下限附近？ | 身高是硬门槛，提前知道才能不把家庭期望押在会被身高挡死的方向上 |
| 既往病史 / 慢性病 | 孩子是否有可能影响特定专业录取的既往病史？ | 病史相关限制最容易被忽略，提前向专业人士确认才能避免临门一脚被退 |

总理由：体检结论一旦出来很难改变，提前自查能让「选科」和「方向倒推」绕开会被体检挡死的路，而不是定科后才发现白走。

具体某专业的体检限报达标标准、退档规则——一律见第一本[第 4 章·招生章程才是硬门槛](../../新疆家长志愿风险识别手册/chapters/04_招生章程才是硬门槛.md)，不在本附录展开。

需复核标注：体检受限专业的具体条目与限报口径（需复核：以当年官方体检指导意见及各校招生章程为准）。

## 2. 军警公安体测与政审方向（提前自查方向是否可能）

| 自查项 | 自查问题（确定没问题 / 可能触线 / 不知道） | 越早查越值钱的理由 |
|---|---|---|
| 体能基础 | 孩子当前体能离军警公安类的体测方向大概是什么差距？ | 体能是少数「越早练越来得及」的项，高一启动和高三临时抱佛脚完全两回事 |
| 政审方向 | 家庭是否了解政治考核大致涉及哪些方面、是否可能存在需提前了解的情形？ | 政审一旦不通过，前面所有准备归零，且无法临时补救，需最早确认 |
| 视力 / 身高 | 是否已对照本附录第 1 节中的视力、身高两项？ | 军警公安对身体门槛叠加要求，任一不达标即整条路关闭 |

注意边界：本附录只帮你判断「这个方向对我家是否一开始就不可能」。体测各项的具体达标数值、政审逐条达标标准、合格 / 不合格如何认定——一律见第一本[第 8 章·专项场景不能套通用模板](../../新疆家长志愿风险识别手册/chapters/08_专项场景不能套通用模板.md)，不在本附录展开。

需复核标注：体测与政审的具体标准、报考条件（需复核：以当年官方招生通知及政治考核要求为准）。

## 3. 医学类色觉 / 视力（单列，因限制最强）

本节与第 1 节合看、不必重复自查；单列是因为医学类体检限制最强、最该提前定位。

| 自查项 | 自查问题（确定没问题 / 可能触线 / 不知道） | 越早查越值钱的理由 |
|---|---|---|
| 色觉与医学专业 | 若孩子有医学方向意向，是否已确认色觉情况？ | 色盲、色弱在医学类多数专业直接受限，提前知道才能避免把医学当默认选项后落空 |
| 既往病史与医学专业 | 是否了解医学类对部分既往病史的限制？ | 医学类体检限制最严，越早自查越能保护「选科 + 方向」两步不踩空 |

医学类各专业的具体录取限制——见第一本[第 4 章·招生章程才是硬门槛](../../新疆家长志愿风险识别手册/chapters/04_招生章程才是硬门槛.md)。

需复核标注：医学类体检限制条目（需复核：以当年官方体检指导意见及各医学院校招生章程为准）。

## 4. 艺体类身体要求（提前自查门槛）

| 自查项 | 自查问题（确定没问题 / 可能触线 / 不知道） | 越早查越值钱的理由 |
|---|---|---|
| 艺术类身体条件 | 若有艺术方向意向，是否了解部分艺术专业（如航空服务艺术与管理）对身高、视力、色觉、体态等的要求？ | 艺体路线集训投入大，先确认身体门槛能避免投入数年后被身体条件挡住 |
| 体育类基础 | 若有体育方向，孩子身体基础是否支持？ | 体育类启动早、训练周期长，越早评估越能避免高三才发现来不及 |

艺体路线该不该提前上车、各自时间窗，见本书第 8 章；具体达标标准见第一本[第 8 章·专项场景不能套通用模板](../../新疆家长志愿风险识别手册/chapters/08_专项场景不能套通用模板.md)。

需复核标注：艺体类身体要求条目（需复核：以当年官方及各校招生简章为准）。

## 5. 外语语种（容易被整条漏掉的早筛项）

| 自查项 | 自查问题（确定没问题 / 可能触线 / 不知道） | 越早查越值钱的理由 |
|---|---|---|
| 语种限制 | 孩子高考外语是否为非英语语种？是否了解部分专业 / 院校只招英语语种考生？ | 语种限制在出分后才发现会直接卡死整批专业，提前知道才能在选科与方向阶段纳入考虑 |

具体哪些专业 / 院校限语种、如何逐条核验——见第一本[第 4 章·招生章程才是硬门槛](../../新疆家长志愿风险识别手册/chapters/04_招生章程才是硬门槛.md)。

需复核标注：语种限制条目（需复核：以当年各校招生章程语种条款为准）。

## 6. 户籍 / 学籍年限类资格（独立一票否决项 · 最优先自查）

这一类与身体条件并列，但更隐蔽：它不看孩子能力，只看「时间够不够」。专项计划（如国家专项、地方专项、南疆相关计划）常要求户籍满一定年限、学籍连续就读一定年限。一旦年限不够，无论分数多高都不能报，且过期补不回——这是全附录最该最早确认的一项。

| 自查项 | 自查问题（确定没问题 / 可能触线 / 不知道） | 越早查越值钱的理由 |
|---|---|---|
| 户籍年限 | 孩子户籍迁入本地是否已满规定年限？（部分专项计划对户籍迁入年限设有硬性下限，差一天即不可报，具体年限以当年官方报考条件为准·需复核） | 户籍年限按时间累计，发现不够时往往已无法补足，必须在初升高就确认 |
| 学籍连续年限 | 孩子是否在符合要求的学校连续就读规定年限？（部分专项对高中学籍连续就读年限设有硬性下限，具体年限以当年官方报考条件为准·需复核）是否有转学中断风险？ | 学籍连续性一旦中断难以恢复，转学前就该评估是否会破坏专项资格 |
| 户籍学籍一致性 | 孩子的户籍地与就读地是否符合所关注专项计划的「户籍 + 学籍双满足」要求？ | 双满足是专项资格的常见硬条件，越早发现错配越有调整空间 |
| 南疆 / 国家专项敏感度 | 是否有意向走南疆或国家专项？这类计划对户籍学籍年限尤其敏感，是否已就这条单独确认？ | 这类计划资格门槛最硬，提前确认才能决定要不要把它纳入三年布局 |

一票否决提醒：上表任一项答「可能触线」或「不知道」，都不要自行判断「能不能享受专项」，直接进入人工复核。专项资格逐条核验、确认录取流程，见第一本[第 8 章·专项场景不能套通用模板](../../新疆家长志愿风险识别手册/chapters/08_专项场景不能套通用模板.md)。

需复核标注：户籍满年限、学籍连续年限、专项计划报考资格的具体年限与口径（需复核：以当年新疆教育考试院及专项计划官方发布为准）。

## 7. 何时必须人工复核

凡命中以下任一情形，停在「可能触线 / 不知道」，转人工复核，不要照本附录自行下「能走 / 不能走」的结论：

- 任何身体条件项（色觉、视力、身高、病史、语种）答「可能触线」或「不知道」。
- 户籍 / 学籍年限任一项答「可能触线」或「不知道」，尤其涉及南疆 / 国家专项。
- 想把某条受身体或年限影响的路线纳入三年布局，但拿不准是否一开始就被挡死。

咨询入口与服务边界见 [附录 M：服务边界与咨询入口](附录M_服务边界与咨询入口.md)。具体达标标准、逐条核验，见第一本《别把孩子的分数浪费在志愿表里》[第 4 章](../../新疆家长志愿风险识别手册/chapters/04_招生章程才是硬门槛.md)、[第 8 章](../../新疆家长志愿风险识别手册/chapters/08_专项场景不能套通用模板.md)。

本附录只做一件事：在出分前把「可能被身体或年限挡死的路」标出来。标出来之后怎么核验、达不达标，交给人工复核与第一本。

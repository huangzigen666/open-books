# 附录 K：路径分流自测与家庭信息采集卡

本附录结论：**这张卡只帮家庭粗判现在该先盯哪一到两个里程碑，具体走哪条路、怎么排节奏，一律到此为止并转人工复核；别把它当成家庭升学方案。**

## 怎么用这张卡

先用[第1章：三年时间轴总览 + 不可逆里程碑地图](../chapters/01_三年时间轴与里程碑总览.md)定位孩子现在站在哪一段，再做下面的自测。自测只做方向粗判，不给结论；想真正评估，就带着后面的家庭信息采集卡，转到[附录M·服务边界与咨询入口](附录M_服务边界与咨询入口.md)做一对一复核。

## 路径分流自测

请只勾选，不要在这里推导方案。

- [ ] 孩子目前所处阶段：初升高 / 高一 / 高二 / 高三上 / 接近志愿季。
- [ ] 选科还没有定，或已经定了但没有复核过目标方向限制（需复核：以当年官方发布为准）。
- [ ] 家庭对孩子未来专业大方向只有模糊想法，还没有倒推到学科承受力与限制条件。
- [ ] 家庭正在考虑普通录取之外的其他可能，但不知道是否适合孩子。
- [ ] 孩子可能涉及身体条件、语种、专项资格、户籍/学籍年限等限制项（需复核：以当年官方发布为准）。
- [ ] 家里还没有持续记录成绩趋势、单科变化、孩子本人想法和家庭底线。
- [ ] 已经临近交棒志愿季，但材料、资格、底线、目标方向还没有收口。

## 需复核与官方优先声明

选科要求、专项资格、身体条件限制、语种限制、户籍/学籍年限等都会随政策、院校、批次和年度变化（需复核：以当年官方发布为准）。这张卡不能替代新疆教育考试院、招生高校和当年官方文件，也不能用群聊截图、短视频或 AI 生成内容替代官方核验。

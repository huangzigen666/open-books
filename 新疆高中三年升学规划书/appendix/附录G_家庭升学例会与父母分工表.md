# 附录 G：家庭升学例会与父母分工表

本附录结论：**三年规划不是某一个家长一个人的事。把例会开起来、把分工写下来，比谁更懂政策更重要；这份模板只给空白框架，谁管哪条线由你家自己定。**

## 1. 使用原则

- 分工是为了有人盯，不是为了分输赢。
- 例会留痕是为了下次能复盘，不是为了考核孩子。
- 孩子是当事人，不是旁听者。涉及他本人的决定，要让他在场。
- 空白处由家庭自己填，本附录不替你家分配角色。
- 讨论不等于拍板。涉及选科、路线、复读等个性化判断，必须人工复核。

## 2. 家庭升学例会议程模板

### 频率建议

常规阶段固定开一次。

遇到里程碑节点、学校重要通知、官方政策更新、家庭意见冲突时，临时加开。里程碑包括选科定科、方向倒推、路线分流、资格早筛、院校梯度初定，详见 [附录B·三年决策里程碑日历](附录B_三年决策里程碑日历.md)。

### 固定议题清单

- [ ] 上次待办回顾：哪些完成了，哪些没有完成。
- [ ] 本阶段必做事项与截止信号：对照本书时间轴。
- [ ] 信息更新：按官方 / 需复核 / 小道消息分级，见 [附录F·信息台账与三级信源标注模板](附录F_信息台账与三级信源标注模板.md)。
- [ ] 需要请教专业人士的问题：只记录问题，不现场硬拍板。
- [ ] 孩子本人想说的：学习压力、兴趣变化、不能接受的底线。
- [ ] 下次待办：写清负责人、材料、复核状态。

### 留痕方式

会议纪要至少保留这些要素：

- 日期：____
- 在场人：____
- 本次决定：____
- 依据来源：已查 ____ / 需复核 ____
- 下次会议或复核时间：____

纪要给家庭自己复盘用，不外传，不用来追责孩子。

## 4. 不替孩子包办

父母分工，分的是“谁盯哪条线”，不是“谁替孩子决定人生”。

选科、路线、复读、转轨这些决定，书里只给框架和自查问题。个性化判断要人工复核，入口见 [附录M·服务边界与咨询入口](附录M_服务边界与咨询入口.md)。

最终决定属于孩子和家庭。本附录不替你家拍板。

## 5. 官方优先

例会里涉及的政策、时间节点、选科要求、资格门槛、户籍/学籍年限，一律以新疆教育考试院及当年官方发布为准；需复核项必须标清楚：需复核（以当年官方发布为准）。

## 2. 本章定位（一句话）

这一章把「家庭心态与节奏」从软话题写成升学规划的底座：管理三年里的焦虑、消化成绩波动、化解攀比，讲的是**节奏与分寸**——让前面 1-11 章的所有规划能在一个不崩的家庭里被持续执行下去，而不是被高压拖垮。

## 3. 必含结构（章节=CODEX 第三节八段模板，逐段写满）

按八段模板写，标题层级与第 6 章/第 7 章一致（一级标题 + `## 1.`～`## 8.` + `## 本章自查`）：

1. **本章结论**（紧接标题，第一段加粗，格式 `本章结论：**……**`，冒号在加粗外）：这一章防的是「用三年高压、反复比较、把焦虑转嫁给孩子，最后把亲子关系和孩子继续走下去的意愿拖垮」的错；这类损伤比选错科更难修复，且常常是不可逆的。
2. **你家现在站在哪一段**（时间轴定位）：和第 7 章一样说明——这一章没有固定学期，是贯穿初升高暑假到高三的一条副线；任何阶段只要家庭氛围开始紧绷、对话只剩成绩，就是该读这一章的时候。
3. **这一学期/这三年家庭该做什么**（1-3 件必做事项）：把「经营家庭氛围」落成可操作动作，例如①约定家庭里「谈成绩」和「过日子」分开的边界与频率；②建立成绩波动的解读机制（看趋势不看单次，看投入不只看名次）；③把情绪管理纳入第 3 章已定的父母分工（谁主要管情绪、出现什么信号要叫停）。其中至少一件用「里程碑四件套」微结构写：**时间窗 + 前置准备 + 一次家庭决策 + 一份留痕**（如「定一次家庭节奏公约」可这样写）。注意：本章不命中那五个标准里程碑，**写成「三年心理节奏不可逆点」这条贯穿提醒即可，不要硬塞五个里程碑名**。
4. **不可逆点提醒**：用「可补救的遗憾 vs 不可逆的错过」二分法。可补救：一次考砸后的低落、一段时间的紧张、偶尔的争吵。不可逆/极难修复：长期高压下孩子关闭沟通、对升学彻底失去意愿、亲子信任崩塌——明确点出「这条是前言、第 1 章、第 3 章先点破、本章展开的那条贯穿不可逆点」，且它比选错科更难修复。

## 4. 红线（违反即不通过，本章写死）

- **讲节奏与分寸，反贩卖焦虑、反催补习**：全章基调是「松弛、可持续、把家庭氛围当资源经营」，**严禁用「不抓紧就来不及」「别人都在补/不补就落后」「现在不拼三年白费」之类制造或放大焦虑的话术**；不得直接或暗示性地建议加补习班、加训练量、延长学习时间来解决焦虑。本章是给高压降温的，不是再添一把火。
- **不替家庭做决定（不替代一对一）**：只给「框架 + 判断维度 + 自查 + 何时必须人工复核」，**绝不给「你家该不该减压、要不要停某个班、孩子是不是抑郁、要不要休学」这类结论**；任何涉及孩子心理状态的判断一律停在「观察维度 + 出现 X 信号请找专业人士」。
- **结论先行且加粗**：开篇 `本章结论：**……**`，冒号在加粗外；不写玄学、不承诺结果（禁用「保上/内部名额/百分百/玄学」等表达）。
- **AI 降级 / 不拿工具替代专业判断**：可提 AI/量表只能帮整理和提示，**不能替代对孩子心理状态的专业诊断与援助**；心理危机信号必须导向真人专业援助。
- **不编造任何具体数据**：不写具体年限数字、分数、位次、报名时间、达标标准、心理量表分值；不写「满 X 年/连续失眠 X 天即异常」这类伪标准；时间一律用「相对学期段 + 事件信号」表达（如「一次大考后情绪长期不回升」）。本章无官方复核项，**不要逐句贴「需复核」长句，避免标签疲劳**。
- **案例必须脱敏或合成**：任何举例不得含可识别的真实学生/家庭/学校/成绩信息，用合成化的「有的家庭……」笔法。
- **不越界写第一本内容**：本章基本不触志愿季；若确需提及「出分后该不该复读」这类情绪高发场景，只一句话指向第一本[《别把孩子的分数浪费在志愿表里》第 7 章·低分段不是没路](../../新疆家长志愿风险识别手册/chapters/07_低分段不是没路.md)（复读评估在第一本第 7 章），**不在本章展开复读/志愿内容**，也不要指向不存在的「复读专章」。

## 2. 本条目定位（一句话）

本附录把高中三年里 **5 个做错/错过就回不了头的里程碑**（选科定科 / 方向倒推 / 路线分流 / 资格早筛 / 院校梯度初定）的「最佳布局窗口」与「截止线」排成一张可对照的日历，让家长一眼定位「现在最该盯哪个里程碑、哪个窗口正在关闭」——**但全表只给相对节奏与窗口结构，不给任何确定日期或年限数字，所有时间项一律以当年官方发布为准**。

## 3. 必含结构（清单页体例：本附录结论加粗 + 分节 + 表格/清单；不套八段）

按以下顺序写，对齐第一本附录 A/D 的密度：

1. **本附录结论**（开头加粗一句，格式 `本附录结论：**……**`，冒号在加粗外）：高中三年只有 5 件做错回不了头的事；这张日历只帮你定位「窗口什么时候开、什么时候关」的相对节奏，**任何示例时间都不是确定值，必须按当年官方发布逐项复核**，别把这页当成可照抄的报名表。
2. **怎么读这张日历**（使用说明，3-5 条）：①每个里程碑分「最佳布局窗口（越早越便宜）」与「截止线（过了补不回 / 代价陡增）」两列看；②时间一律是「相对学期段 + 信号」描述，不是具体月份/日期；③**表中任一时间项都要按当年官方发布复核**；④先用第 1 章时间轴定位自己站在哪一段，再回本表对应里程碑；⑤个性化「你家该盯哪个、要不要走某条路」不在本表，走人工复核。
3. **五个里程碑总览表**（一张主表，含**强制需复核表头**，见第 4 节红线）：每行一个里程碑，列建议为「里程碑 / 最佳布局窗口（相对学期段） / 截止线信号 / 错过代价 / 对应章节」。对应章节用规范名引用（选科定科→第 6 章、选科预研→第 5 章、资格早筛→第 7 章、院校梯度初定→第 9 章/第 13 章、方向倒推→第 1 章/第 4 章）。
4. **「路线分流」里程碑专节**（本附录的重点差异化内容）：把竞赛、强基、综合评价三者**各自单列**为一张子表或三张并列小表，列「适配前提一句话 / 最佳启动窗口（相对学期段）/ 截止线信号 / 官方查询入口（写『以教育部、目标高校及新疆教育考试院当年发布为准·需复核』）」。必须显式写明：**这三者不是高二一次性决策——竞赛与综评高一就要启动持续积累，强基高三报名但靠的是前期积累**；不写具体报名月份/具体年限/具体校测规则（那些进第 8 章与第一本）。
5. **「资格早筛」里程碑提示**（一小节，不展开核验细则）：点明户籍/学籍年限类资格「越早确认越好、过期补不回」，最佳窗口是初升高/高一就自查；**具体年限数字与达标标准一律不写**，标「（需复核：以当年官方发布为准）」并指向附录 D 与第 7 章；逐条核验体检/政审/体测达标标准一句话指向第一本《别把孩子的分数浪费在志愿表里》招生章程章。
6. **空白可填「我家里程碑定位卡」**（家长可操作清单 / 可复制框架）：给一张留空表或代码块，让家长自己填「孩子现在所处学期段 / 本阶段正在打开或关闭的窗口 / 已查官方来源 / 待复核项 / 下一步动作」——只给框架，不替家长填结论。
7. **官方优先与需复核声明**（结尾一段）：本表所有时间窗、截止线、启动窗口均为**相对节奏示意**，政策、报名时间、选科要求、资格门槛、户籍学籍年限一律以新疆教育考试院、教育部及目标高校当年官方发布为准；遇示例时间与官方不一致，以官方为准。
8. **交棒与延伸链接**（结尾）：志愿表排序、冲稳保、章程逐条核验、出分后复读决策不在本表，指向第一本；个性化拍板走附录 M 咨询入口。链接见第 5 节。

## 4. 红线（违反即不通过，本条目硬约束）

- **【最重要】表头与每一行的时间项都必须强制标注「时间项均以当年官方发布为准·需复核」。** 具体做法：① 主表与路线分流子表的**表头**里就要带「（时间项均以当年官方发布为准·需复核）」这类整表声明；② 凡出现「最佳布局窗口 / 启动窗口 / 截止线」的**每一行单元格**，文字本身也要让家长读出「这是相对示意、需复核」，绝不能让家长把示例时间当确定值照抄。
- **不提供任何确定日期或年限数字**：禁止写「X 月 X 日」「提前 2 年」「户籍满 3 年 / 学籍连续 3 年」等具体数字当作结论；一律用「相对学期段 + 信号 + （需复核：以当年官方发布为准）」表达，具体数字指向官方与附录 D/第 7 章。
- **不替代一对一**（CODEX 执行指令第七节第 1 条）：只排「窗口什么时候开关」的通用节奏，不给「你家该盯哪个里程碑、该不该走竞赛/强基/综评、该选哪三科」的结论。**强自检**：若家长能照本表抄出一份无需任何人工复核的完整三年时间表，就是写过头了，收回到「相对窗口 + 需复核」层。
- **不写玄学 / 不承诺结果**：禁用「押名校」「保过」「抢跑必赢」「内部名额」「百分百」等表达。
- **不编造官方数据**：不杜撰报名时间、选科要求、资格门槛、年限、校测规则。
- **不越界写第一本内容**：章程逐条核验、冲稳保、72 小时、出分后复读决策一律只做一句话指引，指向第一本，不在本表展开。
- **路线分流三者必须分列**：竞赛、强基、综评不得合并成「高二一次性决策」一行，否则不通过。

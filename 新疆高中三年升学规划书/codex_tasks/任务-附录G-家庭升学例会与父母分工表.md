## 2. 本附录定位（一句话）

这是第 3 章的配套工具页：给家庭一份**空白的「家庭升学例会」议程模板**（频率、议题清单、留痕方式）和一张**空白的父母分工表**（信息/钱预算/情绪沟通/对接专业人士四条线），让规划这件事在家庭里「有人负责、有记录、不互相甩锅、不替孩子包办」——**只给空白框架和模板，由家庭自己填，本附录不替任何家庭分配角色、不下结论。**

## 3. 必含结构（附录＝清单页/模板页体例，**不套八段**）

按「本附录结论（加粗）→ 分节 → 表格/空白模板」组织，参照第一本附录 A/D 的密度。建议分节：

1. **本附录结论**（加粗一句，格式 `本附录结论：**……**`，冒号在加粗外）：例如——三年规划不是某一个家长一个人的事；把例会开起来、把分工写下来，比谁更懂政策更重要；这份模板只给空白框架，谁管哪条线由你家自己定。
2. **使用原则**（3-5 条短句）：例如①分工是为了「有人盯」，不是为了分输赢；②例会留痕是为了下次能复盘，不是为了考核孩子；③孩子是当事人不是旁听者，涉及他本人的决定要他在场；④空白处由家庭自己填，本附录不替你家分配。
3. **「家庭升学例会」议程模板**：
   - **频率建议**（给区间和触发点，不写死）：如「常规每月一次 + 每个里程碑/重大政策更新时临时加开」（里程碑指选科定科、方向倒推、路线分流、资格早筛、院校梯度初定，详见 `附录B_三年决策里程碑日历.md`）。
   - **一张空白议程表**（Markdown 表格，列如「议题 / 谁来讲 / 需要的材料 / 本次结论 / 谁跟进 / 截止」，行内容留空或用占位符 `____`，给 1-2 行示意即可，不替家庭填实质内容）。
   - **固定议题清单**（空白勾选/填空式）：如「上次待办回顾、本阶段必做事项与截止信号、信息更新（官方/需复核/小道消息分级，详见 `附录F_信息台账与三级信源标注模板.md`）、需要请教专业人士的问题、孩子本人想说的、下次待办」。
   - **留痕方式**：给「会议纪要要素」空白清单（日期、在场人、本次决定、依据来源含已查/需复核、下次时间），强调纪要是给「家庭自己复盘」用，不外传。
4. **父母分工表**：一张空白 Markdown 表格，**四条线写死为行**（信息、钱/预算、情绪/沟通、对接专业人士），**谁负责一列全部留空 `____` 由家庭填**。列建议：「职责线 / 主要负责（留空）/ 备份（留空）/ 这条线要做什么（可给中性提示）/ 留痕在哪」。四条线的「要做什么」只写中性职责描述，例如：
   - 信息：盯官方信源与时间节点、维护信息台账（指向 `附录F_信息台账与三级信源标注模板.md`）。
   - 钱/预算：盘点三年常规与特殊路线成本、设预算底线（指向 `附录I_升学预算与资源规划表.md`）。
   - 情绪/沟通：维护家庭氛围、做孩子与另一方家长之间的缓冲。
   - 对接专业人士：判断何时该找一对一、准备咨询材料（指向 `附录M_服务边界与咨询入口.md` 与 `附录L_交棒清单.md`）。
5. **不替孩子包办的提醒**（短小一节）：分工分的是「父母之间谁盯哪条线」，不是「替孩子把人生决定包了」；涉及选科/路线/复读的最终拍板，框架在书里、个性化判断要人工复核（指向 `附录M_服务边界与咨询入口.md`），但**决定权属于孩子和家庭，本附录不替你家拍板**。
6. **官方优先 / 需复核声明**（一句）：例会里涉及的政策、时间节点、选科要求、资格门槛、户籍/学籍年限，一律以新疆教育考试院及当年官方发布为准，需复核项标注清楚（需复核：以当年官方发布为准）。

> 表格里凡是「具体怎么填」的实质内容一律留空或用 `____` 占位；示意行可放中性、非个性化的样例文字，但不得出现替某类家庭分配角色、或替家庭得出选科/路线结论的内容。

## 4. 红线（违反即不通过，写死本条目硬约束）

- **只给空白框架/模板，不替家庭分配、不下结论**：父母分工表「谁负责」列必须留空；不得出现「爸爸应管钱、妈妈应管情绪」之类替家庭指派的写法。
- **不替代一对一**：涉及选科/路线/复读的拍板一律停在「框架＋找人工复核」，指向 `附录M_服务边界与咨询入口.md`；不给「你家该选哪三科/走哪条路/要不要复读」的结论或组合→专业门类映射。
- **不贩卖焦虑**：例会与分工是为了「有人盯、能复盘」，不是制造对立或施压；不写催家长给孩子加补习、不写「不开例会就会落后」之类恐慌话术；明确孩子是当事人不是被考核对象。
- **不写玄学/不承诺结果**：禁用「押名校」「保过」「内部名额」「百分百」等表达。
- **不编造官方数据/时间**：议程里涉及政策/时间/资格的占位一律标「需复核（以当年官方发布为准）」或留空，不杜撰具体节点。
- **案例脱敏**：任何示意文字均为脱敏/合成，不含可识别学生/家庭信息。

## 2. 本附录定位（一句话）

把七个学期段做成**一页速查表**：每段一行，给「必做事项 + 截止信号 + 错过代价」，让家长一眼定位「现在站在哪一段、这一段不动手会丢什么」，作为全书脊柱、第 1 章的随身底图；最后一段（交棒志愿季）只做交棒指引，不展开志愿内容。

## 3. 必含结构（清单页体例：本附录结论加粗 + 分节 + 主表 + 用法说明；**不套八段模板**）

按以下顺序写：

1. **一级标题**保留 `# 附录 A：三年关键窗口时间轴总表`。
2. **本附录结论**（加粗一句，格式 `本附录结论：**……**`，冒号在加粗外）：高中三年的被动不是出分那天才发生的，是七个学期段一连串「该动手没动手」攒出来的；这张表只帮你定位「现在这一段必做什么、错过补不回什么」，**所有时间以当年官方发布为准**。
3. **怎么用这张表**（短小一节，3-5 句）：先在表里找到孩子现在所处的学期段→看「必做事项」清这一段的活→看「截止信号」判断有没有快过窗口→看「错过代价」分清这是「可补救的遗憾」还是「不可逆的错过」；并提示「具体时间窗见 [附录B·三年决策里程碑日历](附录B_三年决策里程碑日历.md)，本表只给段位与代价，不给日期」。
4. **三年关键窗口主表**（一张 Markdown 表，七行，表头建议 `学期段 | 这一段必做事项 | 截止信号（需复核） | 错过代价`）。七行=七学期段，顺序固定：
   - 初升高暑假
   - 高一上
   - 高一下
   - 高二
   - 高三上
   - 高三下
   - 交棒志愿季
   每行写法约束：
   - 「必做事项」给 1-3 件**框架级**动作（如「建成绩档案起点」「做选科×专业限制对照」「户籍/学籍年限第一次自查」「资格早筛」「特殊路线要不要上车的提前评估」），**不给个性化结论**（不写「该选哪三科/走哪条路」）。
   - 「截止信号（需复核）」写**事件型信号**而非日期（如「学校要求确定选科组合前」「专项报名/政治考核启动前」「特殊招生最后报名/确认节点前」），每格行内或表头统一标 **（需复核：以当年官方发布为准）**。
   - 「错过代价」一句话点明丢什么、是否不可逆；命中 5 里程碑的段（选科定科、方向倒推、路线分流、资格早筛、院校梯度初定）要让代价能呼应「可补救 vs 不可逆」。
   - 交棒志愿季这一行：「必做事项」只写「把三年档案打包移交、确认边界」；「错过代价」写「三年准备在出分后断档」；**不在本格写冲稳保/章程核验/填报**。
5. **三段贯穿提醒**（结论先行的小节，可加粗领起，2-4 句）：
   - 全表时间项一律以官方当年发布为准（重申）；
   - 五个不可逆里程碑里，「路线分流」不是高二一次性决策——竞赛与综评高一就要启动持续积累、强基高三报名但靠前期积累，**这三者时间窗各自不同（均需复核），详见 [附录B·三年决策里程碑日历](附录B_三年决策里程碑日历.md)**；
   - 一条「三年心理节奏不可逆点」提示：长期高压把亲子关系/孩子意愿拖垮，比选错科更难修复（点到即止，详见 [第12章·亲子心态与节奏](../chapters/12_亲子心态与节奏.md)）。
6. **交棒指引**（独立一小节，对应主表最后一行）：明确「出分后到填报志愿那几天不在本书范围」，去向用规范相对路径——本书内指 [第13章·交棒志愿季](../chapters/13_交棒志愿季.md)，跨书指第一本 [《别把孩子的分数浪费在志愿表里》](../../新疆家长志愿风险识别手册/README.md)。

## 4. 红线（违反即不通过）

- **全表所有时间项均标「（需复核：以当年官方发布为准）」**，可在表头「截止信号」列统一声明 + 关键格内复述；**不提供任何确定日期/月份/具体报名时段/分数线/位次**。
- **不替代一对一**：必做事项只停在「框架 + 自查 + 需复核」，**严禁出现「该选哪三科 / 该走哪条路 / 该报哪所」的结论**，也不放「组合→专业门类」映射。强自检：若家长能照这张表抄出一份无需任何人工复核的完整三年方案，就是写过头了，收回到框架层。
- **不写玄学/不承诺结果**：禁用「保上名校」「内部名额」「低分包上」「抢跑必赢」「百分百」等表达。
- **不编造官方数据**：不杜撰任何分数线、位次、选科要求、资格门槛、户籍学籍年限、时间节点。
- **志愿季只交棒、不展开**：交棒志愿季那一段及交棒小节，不写冲稳保、章程逐条核验、投档退档、填报前 72 小时——一律指向第一本与本书第 13 章。
- **不与附录 B 重复**：本表给「段位 + 必做 + 截止信号 + 代价」，**不给里程碑的具体最佳布局窗口表**（那是附录 B）。
- **案例脱敏**：如举例须脱敏或合成，不含可识别学生/家庭信息。

## 2. 本章定位（一句话）

高一下学期、3+1+2 选科定科——**全书最大的不可逆决策点**。本章帮家庭建立「怎么判断、要问自己什么、什么时候必须找人复核」的框架，**不替任何一个孩子定哪三科**。

- 命中里程碑：**选科定科（最大不可逆点）**。
- 防不可逆：别犯「选错科直接堵死整类专业、想改又过了窗口」的不可逆错，别只凭一时成绩或跟风定科。

## 3. 章节内统一结构（八段，沿用 CODEX_执行指令.md 第三节）

1. **本章结论**（加粗一段，结论先行）：这章防什么错 + 错过什么。
2. **你家现在站在哪一段**：时间轴定位，1–2 句（高一下，定科窗口将到）。
3. **这一学期家庭该做什么**：1–3 件必做事项；选科定科这件不可逆事项用「里程碑四件套」写——**时间窗 + 前置准备 + 一次家庭决策 + 一份留痕**。
4. **不可逆点提醒**：用「可补救的遗憾 vs 不可逆的错过」二分法，讲清选错科为什么是「事故」级、纠错窗口与代价。
5. **家长可操作清单**：选科家庭决策会的开法 + 判断维度（成绩趋势/学科兴趣/专业方向/特殊路线门槛/抗风险）+ 自查问题。**只给空白框架和问题，不给结论。**
6. **需复核标注**：见第 4 节硬护栏。
7. **何时必须人工复核**：明确指出选科最终拍板必须人工复核，指向咨询入口（附录 M）。
8. **本章自查**：`- [ ]` 复选框清单（沿用第一本格式）。

## 4. 第 6 章硬护栏（最高优先级，违反即不通过）

- **只给「对照方法 + 判断维度 + 自查问题」。严禁给出任何「具体三科组合 → 具体专业门类」的完整映射结论表**（例如「物化生=可报 X / 关闭 Y」这类可照抄的总表）。需要这种对照时，只教**方法**并指向附录 C 的抽样示例与官方资料。
- 任何「某省/某校某专业要求选某科」「选物理/历史打开或关闭哪类专业」「选物理可报 XX% 专业」类具体陈述，**一律行内标「（需复核：以当年官方发布为准）」**并指向官方——教育部选考科目要求有新旧版本、各校执行逐年不同。
- **整章末尾强制收口**：明确写出「最终选哪三科，必须结合孩子具体情况人工复核，本书不替你拍板」，并指向咨询入口。
- **强自检红线**：写完回读一遍——若一个家庭照这章就能产出「无需任何人工复核的最终选科结论」，就是写过头了，必须收回到框架层。

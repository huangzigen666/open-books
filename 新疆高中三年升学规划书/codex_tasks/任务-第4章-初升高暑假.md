## 2. 本条目定位（一句话）

中考后到高一开学这个被普遍浪费的窗口里，把家庭该完成的四件起步动作（学科衔接、成绩建档起点、选科/路线第一次预判、户籍/学籍年限第一次自查）排清楚，命中「方向倒推」里程碑的**起步**，让家长在不犯结论的前提下先把档案与早筛的「第一块砖」放下。

## 3. 必含结构（章节=八段模板，严格按 CODEX_执行指令 第三节）

按以下八段写，标题层级与第一本一致：

1. **本章结论**（加粗一段，结论先行）：格式 `本章结论：**……**`（冒号在加粗外）。点明本章防什么错（把暑假当纯放松、错过起步窗口）+ 错过什么（成绩档案起点、衔接、第一次路线预判、户籍/学籍年限第一次自查）。
2. **你家现在站在哪一段**：时间轴定位 1-2 句——「中考后到高一开学，三年脊柱的第 0 段，现在做的是‘起步动作’不是‘定结论’」。
3. **这个暑假家庭该做什么**：1-3 件必做事项。其中**「成绩建档起点」和「户籍/学籍年限第一次自查」两件按里程碑四件套微结构写**：**时间窗 + 前置准备 + 一次家庭决策 + 一份留痕**（参 CODEX_执行指令 第三节第 3 点）。四件事建议覆盖：
   - 学科衔接（初高中难度断层的认知与节奏铺垫，不写具体补习课表、不催加补习）
   - 成绩建档起点（引用附录 H，把「第一份档案」建起来——这是方向倒推的数据底座）
   - 选科/路线第一次预判（引用附录 K，**只做粗预判方向、找到要重点关注的 1-2 个里程碑，不给选科结论、不给具体路线名**）
   - 户籍/学籍年限第一次自查（**只做「第一次确认是否可能影响专项资格」**，标需复核，指向第 7 章/附录 D 做系统早筛）
4. **不可逆点提醒**：本章命中「方向倒推」里程碑（起步阶段）。用「可补救的遗憾 vs 不可逆的错过」二分法分级：衔接没做好/档案晚建几周→可补救的遗憾；户籍/学籍年限这类**过期补不回**的项若到此刻仍无人确认→指向不可逆早筛（但本章只做「拉响第一次自查」，系统排查交第 7 章）。
5. **家长可操作清单**：给空白可填框架 / 自查问题（不给个性化结论）。可用一张「暑假起步四件事 · 状态自查表」（列：事项 / 当前状态 / 下一步 / 去哪做），表格密度对齐第一本第 1 章那张风险表。
6. **需复核标注**：凡涉及户籍/学籍年限、专项资格门槛，一律行内标「（需复核：以当年官方发布为准）」。不写任何具体年限数字、具体专项名下的具体门槛。
7. **何时必须人工复核**：点明本章哪些判断必须找专业人士——尤其户籍/学籍年限「是否可能触线」一旦自查出疑点、以及路线预判出现分叉时，指向咨询入口（`../appendix/附录M_服务边界与咨询入口.md`）。
8. **本章自查**：`- [ ]` 复选框清单（沿用第一本格式），覆盖四件起步动作是否各自落地 + 是否留痕 + 户籍学籍年限是否已做第一次自查 + 是否克制住了「现在就替孩子定科/定路线」的冲动。

## 4. 红线（违反即不通过，本条目硬约束写死）

- **户籍/学籍年限：只做「第一次自查」，不展开达标标准。** 本章只回答「是否可能影响专项资格、要不要尽早确认」，**一律标需复核**，并把系统早筛交棒第 7 章 `chapters/07_身体条件与资格早筛.md` 与 `../appendix/附录D_身体条件与资格早筛清单.md`。**不写具体年限数字（如满几年）、不写某专项的具体资格门槛、不写逐条核验流程。** 凡进入「逐条核验体检/政审/专项达标标准」一律一句话指向第一本（户籍/资格类核验详见第一本 `../../新疆家长志愿风险识别手册/chapters/08_专项场景不能套通用模板.md`；招生章程硬门槛详见第一本 `../../新疆家长志愿风险识别手册/chapters/04_招生章程才是硬门槛.md`）。
- **不给选科结论。** 路线/选科只停在「第一次预判 + 粗标方向 + 找到要关注的里程碑 + 需复核/转人工」。**严禁**出现「应该选哪三科」「该走哪条路线」「某组合→某专业门类」的映射或结论。附录 K 的引用要尊重它写死的输出边界（只到「方向粗标 + 转人工」）。
- **不写玄学/不承诺结果**：禁用「押名校」「保上线」「抢跑必赢」「内部名额」「百分百」等表达。
- **不贩卖焦虑、不催补习**：讲起步节奏与取舍，不开补习课表、不点名机构、不写鸡血。
- **不编造官方数据**：不写具体分数线、位次、年限、报名时间、选科要求；不确定一律标需复核或留空白框架。
- **案例必须脱敏或合成**：禁止可识别学生/家庭信息（真实成绩、学校、获奖、户籍轨迹）。
- **强自检红线**：若本章已能让家庭照抄出一份「无需任何人工复核的完整暑假方案 + 选科/路线结论」，就是写过头了，必须收回到框架层。

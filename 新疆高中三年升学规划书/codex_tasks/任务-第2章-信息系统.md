## 2. 本章定位（一句话）

教家庭搭一套能撑满三年、不被谣言牵着走的信息系统：**锁定官方信源 → 用「官方/需复核/小道消息」三级标注法过滤 → 定更新节奏 → 建一张家庭信息台账**。这是贯穿全书的副线工具——后面每一章遇到的政策、时间、选科要求、资格门槛，都靠这套系统去查、去标、去存。

## 3. 必含结构（套用八段模板，CODEX_执行指令第三节）

按八段写，每段落到「信息系统」这个主题：

1. **本章结论**（加粗一段，格式 `本章结论：**……**`，冒号在加粗外）：三年里最值钱的不是某条消息，而是一套能反复用的信息系统——**官方信源锁死、消息分三级标注、按节奏更新、用台账留痕**；信息台垮了，再勤奋的规划都在沙地上盖楼。
2. **你家现在站在哪一段**（1-2 句时间轴定位）：信息系统是横切能力，**不强绑某个里程碑**——无论孩子在初升高暑假还是高三上，越早把信息台立起来越省事；点一句「这章任何年级都该先读」。
3. **这一年家庭该做什么**（1-3 件必做）：①锁定官方信源清单（新疆教育考试院及相关招生高校官网等，**入口处统一标「以官网为准·需复核」**）；②约定三级标注法并落到家庭习惯；③定更新节奏（如固定查官网的频率、关键窗口期加密）。其中「锁定官方信源」这件事可用「里程碑四件套」微结构写：**时间窗（越早越好）+ 前置准备（列出要盯的官方入口类别）+ 一次家庭决策（谁负责盯、多久查一次）+ 一份留痕（写进信息台账）**。
4. **三级标注法**（本章核心知识块，单独成节，用清单或表格）：把任何一条信息进门先贴一个标签——
   - **官方**：来自官网/官方公众号原文、可追溯到发布主体与日期；可作为决策依据。
   - **需复核**：方向可能对，但会逐年变（政策、时间、选科要求、批次、资格门槛、户籍学籍年限），或只看到二手转述；**先按「需复核」存，回官方原文核对后再升级为「官方」**。
   - **小道消息**：群聊截图、短视频口播、家长口耳相传、AI 未经核实的生成内容；**可作为「去哪查」的线索，绝不可作为长期决策依据**。
   并写一句 AI 降级口径：AI 只能帮你把零散信息整理成「待核查清单」，不替你确认真伪，凡 AI 给的结论先打「需复核」。
5. **更新节奏**（一小节）：给「常态频率 + 关键窗口加密」的框架（不写死具体月份/具体频率数字，留给家庭按当年官方日历定）；点明政策/时间会变，所以信息不是查一次就完，要按节奏复查、给旧信息标过期。
6. **家长可操作清单 / 家庭信息台账**：给一个**空白可填的台账框架**（表头示例：信息条目 / 来源链接或出处 / 三级标注 / 记录日期 / 下次复查时间 / 是否影响哪个决策），强调这是简版、完整模板见附录 F。只给框架与自查问题，**不给个性化结论**。
7. **不可逆点提醒 + 何时必须人工复核**：用「可补救的遗憾 vs 不可逆的错过」二分——信息查晚了多数可补，但**错过官方政策/报名/资格确认的时间窗，过期补不回**；并明确：凡涉及选科要求、特殊招生资格、户籍学籍年限、体检限报等会改命的判断，信息系统只负责「查到并标好」，**最终是否触线、能不能走，必须人工复核**，指向附录 M 服务边界与咨询入口（`../appendix/附录M_服务边界与咨询入口.md`）。
8. **本章自查**（`- [ ]` 复选框，沿用第一本格式）：例如——我有没有把官方信源清单写下来；每条信息有没有贴三级标签；有没有定复查节奏；台账里每条有没有来源；AI 给的内容有没有先标需复核。

## 4. 红线（违反即不通过，本章硬约束）

- **官方信源入口统一标「以官网为准·需复核」**：凡列到新疆教育考试院/招生高校官网等入口，必须在入口处带这一标注；**不得编造任何具体官网网址、域名、公众号名称、具体查询页面路径**——只写信源「类别」（如「省级招生考试机构官网」「目标高校本科招生网」），让家庭按当年官方实际入口去找。
- **不写死会变的具体值**：本章不写具体政策条文、具体时间节点、具体频率数字、具体资格门槛；凡会逐年变的一律标「（需复核：以当年官方发布为准）」。
- **不把小道消息当决策依据**：明确写群聊截图/短视频/家长口传/AI 未核实内容只能当线索，不能当长期决策依据。
- **不泄露产品壁垒**：不写问路工作台的字段结构、评分算法、复核规则；台账只给家长能用的简版表格，不暴露任何内部算法或字段设计。
- **不替代一对一**：信息系统只负责「查到、标好、存住」，凡「能不能走/该选哪科/走哪条路」的拍板一律导向人工复核，不在本章给结论。
- **不写玄学**：禁用「押名校」「保过」「内部名额」「内部渠道」等表达。
- **案例脱敏**：如举例，一律脱敏或合成，不含可识别学生/家庭信息（真实成绩、学校、获奖、户籍轨迹）。

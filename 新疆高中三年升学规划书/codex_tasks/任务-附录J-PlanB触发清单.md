## 2. 本附录定位（一句话）

这是一张**「PlanB 提前预警卡」**：把复读、普高转职教、艺体转轨、留学四类 PlanB 各自「何时该坐下来认真评估一次」的**触发信号**和「在那之前先留好哪些后路」的**提前准备方向**摆出来，让家庭别等走投无路才第一次想 PlanB——**但绝不替家庭判断「到底要不要走某条 PlanB」，那一律留人工；出分后该不该复读则属于第一本。**

## 3. 必含结构（清单页体例：本附录结论加粗 + 怎么用 + 四类 PlanB 分节表 + 共同提醒 + 需复核声明 + 交棒/转人工，不套八段）

1. **本附录结论**（加粗一句，格式 `本附录结论：**……**`，冒号在加粗外）：例如——这张卡只帮你认出「该停下来认真评估 PlanB 的信号」并提前把后路留好（如保留学籍、了解政策窗口），**要不要真的走复读/转职教/转轨/留学一律到此为止、转人工复核**；出分后该不该复读、复读能提多少分，见第一本第 7 章。
2. **怎么用这张卡（2-4 句）**：先用第 1 章时间轴定位现在站在哪一段（链接[第1章·三年时间轴总览](../chapters/01_三年时间轴与里程碑总览.md)）；这张卡只帮你识别「触发信号」和「提前准备方向」，不帮你拍板走不走；任何一类真要评估，带着信息转一对一（指向附录 M）。强调：**勾到触发信号 ≠ 该走 PlanB，只代表「该认真评估一次」。**
3. **四类 PlanB 触发与提前准备表**（每类一张小表或一段卡片，**四类齐全**：复读、普高转职教、艺体转轨、留学）。每类只写两列内容：
   - **触发信号（出现哪几种情况，就该坐下来认真评估这条 PlanB 了）**：写成「自查问题/信号描述」，让家长答「像 / 不像 / 不确定」，**不写分数、位次、达标数字**；信号用「相对学期段 + 事件信号」表达（如「孩子在某学期出现持续的、非偶发的状态下滑」「现有路线的前置条件被证实走不通」），不写「满 X 分」「掉到第 X 名」。
   - **提前准备方向（在真正决定之前，先把哪些后路留好 / 哪些政策窗口先了解）**：只给**方向**，如「了解复读的政策窗口与报名资格大致要求（需复核）」「转职教前先了解学籍能否保留/如何衔接（需复核）」「转轨/集训前先看身体门槛与预算量级（指向附录 D、附录 I）」「留学方向先了解语言与时间窗、是否需提前规划课程（需复核）」。**不写「该不该走」「建议走 X」「X 型孩子适合复读」之类结论或倾向。**
4. **共同提醒（3-5 条短句）**：四类 PlanB 的政策、窗口、资格逐年变动；这张卡不含任何「走不走」的结论；留后路（保留学籍、了解窗口）是低成本动作，真正决策必须一对一；不要在焦虑中临时拍板 PlanB。
5. **需复核与官方优先声明**：复读报名资格与政策窗口、转职教/中职衔接与学籍政策、艺体转轨各路线时间窗、留学相关政策与时间窗，一律以当年官方发布为准；**全表同一长句只在表头/表注写一次**，单元格内用短「需复核」或留空，**正文不逐句贴长句**（避免标签疲劳）。统一口径：`（需复核：以当年官方发布为准）`。

## 4. 红线（违反即不通过，本条目写死）

- **只给「触发信号 + 提前准备方向」，结论留人工**：**严禁**出现「你家该不该走复读/转职教/转轨/留学」的结论或倾向，**严禁**「孩子是 XX 情况 → 建议走 XX PlanB」的映射，**严禁**把任一 PlanB 描述成「退路稳妥 / 兜底 / 划算」。命中触发信号只代表「该认真评估一次」，不代表「该走」。**强自检**：若家庭能照这张卡自己拍板走哪条 PlanB 而无需任何人工复核，就是写过头了，收回到「信号 + 方向」两件套。
- **出分后复读决策一律交棒第一本第 7 章**：本附录不写「出分后该不该复读」「复读能提多少分」「复读分数预期」「低分段怎么选」——这些是第一本[第 7 章·低分段不是没路](../../新疆家长志愿风险识别手册/chapters/07_低分段不是没路.md)的事；本附录只停在「出分前/三年中的提前评估与留后路」。链接落点必须是第一本第 7 章，**不要指向第一本不存在的「复读专章」**。
- **政策窗口标需复核、不写具体时间**：复读报名/政策窗口、转职教学籍衔接政策、艺体转轨时间窗、留学时间节点一律**不写具体月份/日期/报名时间/年限/分数/达标标准**；时间一律用「相对学期段 + 事件信号」表达；会逐年变的标`（需复核：以当年官方发布为准）`。

- **不编造数据**：不杜撰具体分数线、提分幅度、复读年限、报名时间、各路线达标标准、留学时间节点；不确定一律标需复核或留空白框架。

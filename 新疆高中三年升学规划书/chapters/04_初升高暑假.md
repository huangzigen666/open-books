# 第四章：初升高暑假——开学前要锁定的几件事

本章结论：**初升高暑假不是拿来把孩子塞满安排的，它的价值是把三年规划的第一块砖放下：完成衔接认知、建立成绩档案起点、做一次选科/路线粗预判、第一次自查户籍/学籍年限是否可能影响专项资格（需复核：以当年官方发布为准）。把这个暑假当纯放松，最容易错过的是后面方向倒推的数据底座。**

## 1. 你家现在站在哪一段

中考后到高一开学，是三年脊柱的第0段。

这一段不负责定结论，只负责起步：把孩子从初中节奏切到高中节奏，把家庭从“等学校安排”切到“开始留痕、自查、复核”。

## 2. 这个暑假家庭该做什么

第一，做学科衔接，但不要把它做成焦虑加码。

初高中断层真实存在。难点不是多刷几张卷子，而是让孩子知道高中学习更看重概念理解、持续复盘和单科承受力。家庭可以做的是：陪孩子识别哪几科进入高中后最可能吃力，约定开学后怎么观察，不要在暑假直接给孩子排一张满到喘不过气的课表。

第二，建立成绩建档起点。

成绩档案不是高三才有用。高一以后要判断选科、方向、路线、目标收口，都离不开趋势。暑假先把第一份档案框出来，后面每次主要考试往里补。

| 四件套 | 家庭要做什么 |
|---|---|
| 时间窗 | 初升高暑假开始，不等第一次高中考试结束 |
| 前置准备 | 收集中考成绩、初中阶段主要考试、孩子明显强弱科、家庭对成绩波动的观察 |
| 一次家庭决策 | 约定以后每次主要考试都记录总分、单科、位次或相对位置、原因判断 |
| 一份留痕 | 建一份成绩趋势档案，入口见[附录H·三年成绩与位次档案模板](../appendix/附录H_三年成绩与位次档案模板.md) |

附录 H 目前承担的是成绩档案入口。你先把档案意识立起来，具体字段后续按模板补齐，不要等到高三才回头补记忆。

第三，做一次选科/路线第一次预判。

这里的“预判”不是替孩子拍板，也不是替家庭收口。它只问：家庭现在大概要重点关注哪些里程碑。

操作很简单：用[附录K·路径分流自测与信息采集卡](../appendix/附录K_路径分流自测与信息采集卡.md)粗标一遍。结果只能停在“方向粗标 + 转人工”。如果自测显示同时命中多个风险，不要在书里继续推导方案。

第四，第一次自查户籍/学籍年限。

这件事要早，不要细。暑假只做一个判断：孩子的户籍/学籍年限是否可能影响后续专项资格（需复核：以当年官方发布为准）。

| 四件套 | 家庭要做什么 |
|---|---|
| 时间窗 | 初升高暑假先查一次，后续有转学、迁户、资格变化时再复核 |
| 前置准备 | 准备孩子户籍、学籍、就读地、是否可能涉及专项资格的基本信息（需复核：以当年官方发布为准） |
| 一次家庭决策 | 只判断“确定没问题 / 可能触线 / 不知道”，不自行判断能不能享受资格 |
| 一份留痕 | 把结论写进家庭信息台账，并标明需要到本书第7章和附录D继续系统早筛 |

系统早筛见[本书第7章·身体条件与资格早筛](07_身体条件与资格早筛.md)和[附录D·身体条件与资格早筛清单](../appendix/附录D_身体条件与资格早筛清单.md)。

## 3. 不可逆点提醒

本章命中的里程碑是：**方向倒推（起步）**。

方向倒推不是现在就把未来定死。它是先把数据、限制、孩子意愿和家庭底线摆上桌，避免高一高二每次遇到选择都临时拍脑袋。

可补救的遗憾有三类：

- 暑假衔接做得不充分，开学后再调整学习节奏。
- 成绩档案晚建几周，后面补上来源和说明。
- 第一次预判不准确，后续通过考试趋势和孩子反馈修正。

不可逆的错过要盯紧：

- 户籍/学籍年限这类时间累计项，到后来才发现可能影响专项资格（需复核：以当年官方发布为准）。
- 家庭一直没有留痕，高三需要判断趋势时只剩模糊印象。
- 还没看见孩子真实承受力，就先替孩子拍板。

本章只负责拉响第一次自查。具体专项资格、招生章程硬门槛、逐条达标标准，不在这里展开；需要进入逐条核验时，转第一本[第8章·专项场景不能套通用模板](../../新疆家长志愿风险识别手册/chapters/08_专项场景不能套通用模板.md)和[第4章·招生章程才是硬门槛](../../新疆家长志愿风险识别手册/chapters/04_招生章程才是硬门槛.md)。

## 4. 家长可操作清单

先填这张表。它不是方案，是起步状态表。

| 事项 | 当前状态 | 下一步 | 去哪做 |
|---|---|---|---|
| 学科衔接 | 已做 / 未做 / 不清楚 | 识别高中后最可能吃力的科目，不把暑假安排塞满 | 家庭讨论 |
| 成绩建档起点 | 已建 / 未建 / 信息不全 | 把中考与初中主要成绩先放进档案 | 附录H |
| 选科/路线第一次预判 | 已粗标 / 未粗标 / 分歧大 | 只标一到两个重点里程碑，不给路线结论 | 附录K |
| 户籍/学籍年限第一次自查 | 确定没问题 / 可能触线 / 不知道 | 可能触线或不知道，就进入人工复核（需复核：以当年官方发布为准） | 第7章 / 附录D |

再问四个问题：

1. 孩子进入高中后，哪一科最可能从“会做题”变成“理解跟不上”？
2. 家里有没有一份以后能持续补充的成绩档案？
3. 我们是不是把“第一次预判”误当成了“最终路线决定”？
4. 户籍/学籍年限是否已经做过第一次自查，并留下依据（需复核：以当年官方发布为准）？

## 5. 需复核标注

本章涉及的户籍/学籍年限、专项资格、选科要求、身体条件限制，都会随政策、学校、批次和年度变化（需复核：以当年官方发布为准）。

本章不写任何具体年限数字、不写某项专项资格的具体门槛、不写逐条核验流程。凡家庭自查结果为“可能触线”或“不知道”，先留痕，再回到官方发布和人工复核。

## 6. 何时必须人工复核

出现下面情况，不要继续自己推：

- 户籍/学籍年限第一次自查结果是“可能触线”或“不知道”（需复核：以当年官方发布为准）。
- 附录 K 粗标结果同时命中多个里程碑，家庭不知道先处理哪个。
- 选科倾向、孩子兴趣、成绩强弱和家庭期待互相冲突。
- 家庭准备因为一次暑假测试或一次亲戚建议，就替孩子拍板。
- 涉及专项资格、身体条件、语种限制、招生章程硬门槛，需要进入逐条核验。

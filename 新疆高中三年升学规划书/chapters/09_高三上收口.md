# 第九章：高三上·从「积累」切换到「收口」

本章结论：**高三上防的是“还在做大方向幻想、材料散在各处、特殊招生最后报名/确认节点漏核”的错。这个阶段错过的不是一次考试，而是过期补不回的收口窗口，前面三年的准备可能在最后一步散掉。**

高三上不是继续无限扩展可能性的阶段。

它的关键词是收口：把能查的材料定稿，把目标方向收窄，把特殊招生最后节点逐项核对，为出分后的志愿季留出清楚、可交接的输入。

## 1. 你家现在站在哪一段

你家站在从“积累期”进入“收口期”的换挡段。

再往前一步，就是出分和志愿季。本书还负责把材料、方向和风险收拾好；志愿表怎么排、冲稳保怎么分，是第一本接棒的事。

## 2. 这一学期家庭该做什么

高三上只做三件事。表中涉及时间、资格、特殊招生节点的内容统一（需复核：以当年官方发布为准）。

| 必做事项 | 时间窗 | 前置准备 | 一次家庭决策 | 一份留痕 |
|---|---|---|---|---|
| 材料与数据定稿 | 高三上，学校与官方信息密集更新后 | 三年成绩档案、选科记录、专业限制线索、资格早筛结论、特殊路线材料 | 确认哪些材料已经可用，哪些仍是待复核，不把群聊截图当依据 | 建一份“已查 / 需复核 / 尚缺”的收口清单 |
| 目标收敛 | 高三上从积累切到收口时 | 孩子的真实成绩趋势、选科边界、家庭偏好、预算底线 | 从“大方向幻想”收窄到按层次粗分的院校方向区间，不列具体志愿表 | 写清保留方向、排除方向、需要第一本接棒的问题 |
| 特殊招生最后报名/确认节点核对 | 特殊招生简章、学校通知或官方入口更新前后 | 可能涉及的路线清单、官方入口、材料状态、孩子意愿 | 只判断“有没有节点、是否仍在窗口内、是否必须人工复核”，不判断最终走不走 | 逐项记录官方入口、当前状态、负责人和复核结果 |

材料与数据定稿，必须回到[第二章·信息系统](02_信息系统.md)和[附录F·信息台账与三级信源标注模板](../appendix/附录F_信息台账与三级信源标注模板.md)。高三上最怕的不是材料少，而是材料多、来源乱、状态不清。

目标收敛只做到“方向区间”。你可以把方向粗分为更想保留、可以接受、暂不考虑、需人工复核几类；不能在本章里排院校顺序，更不能开始做志愿层级清单。

特殊招生最后报名/确认节点，是本章最高危事项。这里也只做核对：有没有这个节点、是否已经过窗口、去哪个官方入口确认、要不要转人工。

## 3. 不可逆点提醒

本章命中的里程碑是**院校梯度初定（启动）**。

可补救的遗憾，是目标方向还不够细、某些材料还需要补来源、家庭偏好还要再谈。这些进入第十三章交棒前仍能继续整理。

不可逆的错过，是特殊招生最后报名/确认节点过期（需复核：以当年官方发布为准）。一旦过了窗口，前面高一、高二为这条路线做过的积累可能当年无法使用。这个错不靠临场努力补，也不靠情绪补。

另一个高风险，是高三上还沉迷“大方向幻想”。到这一步，家庭不该继续问“还有没有更神奇的路”，而该问“哪些路已经有证据，哪些路必须放弃，哪些路只能交给出分后第一本来做”。

## 4. 家长可操作清单

先做材料定稿核对框。

| 材料项 | 当前状态 | 来源等级 | 待复核问题 |
|---|---|---|---|
| 三年成绩档案 | 已查 / 需复核 / 尚缺 | 官方 / 需复核 / 小道消息 | |
| 选科与专业限制线索 | 已查 / 需复核 / 尚缺 | 官方 / 需复核 / 小道消息 | |
| 身体条件与资格早筛结论 | 已查 / 需复核 / 尚缺 | 官方 / 需复核 / 小道消息 | |
| 特殊招生路线材料 | 已查 / 需复核 / 尚缺 | 官方 / 需复核 / 小道消息 | |
| 家庭偏好与底线 | 已达成 / 需讨论 / 有冲突 | 家庭会议纪要 | |

再做目标收敛自查。

- 孩子现在真正能保留的方向有哪些？
- 哪些方向被选科、身体条件、资格或预算挡住？
- 家庭能接受的城市、专业、费用和风险边界是什么？
- 哪些目标只是情绪期待，没有证据支撑？
- 哪些问题必须等出分后由第一本接棒？

最后做特殊招生最后节点核对框。表中节点、入口和状态均需复核：以当年官方发布为准。

| 路线 | 是否涉及 | 官方入口 | 当前状态 | 下一步 |
|---|---|---|---|---|
| 强基 / 竞赛 / 综合评价 | 是 / 否 / 不确定 | 待填 | 在窗口内 / 已过 / 需复核 | |
| 艺术 / 体育 | 是 / 否 / 不确定 | 待填 | 在窗口内 / 已过 / 需复核 | |
| 军警公安 / 医学 / 公费师范 | 是 / 否 / 不确定 | 待填 | 在窗口内 / 已过 / 需复核 | |
| 南疆 / 国家专项 / 预科 | 是 / 否 / 不确定 | 待填 | 在窗口内 / 已过 / 需复核 | |

## 5. 需复核标注

高三上所有会变化的事项，都不要凭旧资料拍板。

特殊招生最后报名/确认节点、出分与志愿时间、资格口径、材料提交要求、目标高校简章口径，均需复核：以当年官方发布为准。同一张表只在表注写完整复核口径，单元格只保留“需复核”或空白，避免把标签贴到失效。

AI 可以帮你整理材料缺口和台账状态，不能替你确认节点是否还来得及，也不能替你判断该不该保留某条路线。

## 6. 何时必须人工复核

出现下面任一情况，直接走[附录M·服务边界与咨询入口](../appendix/附录M_服务边界与咨询入口.md)：

- 不确定某条特殊招生路线最后报名/确认节点是否还来得及。
- 目标收敛与选科、资格、身体条件或预算对不上。
- 特殊招生材料临门一脚拿不准。
- 家庭想保留多条路线，但孩子承受力和时间精力明显冲突。
- 家庭已经开始把“先列个志愿表看看”当成缓解焦虑的方法。

边界写清楚：志愿表整体怎么填，见[第一本第1章·志愿填报的第一性原理](../../新疆家长志愿风险识别手册/chapters/01_志愿填报的第一性原理.md)；冲稳保怎么排序和分配，见[第一本第3章·冲稳保不是万能药](../../新疆家长志愿风险识别手册/chapters/03_冲稳保不是万能药.md)。出分后到填报阶段的整体方法，交棒[第一本《别把孩子的分数浪费在志愿表里》](../../新疆家长志愿风险识别手册/README.md)。

本章只是把高三上该收的东西收好。下一步出分后的打包移交，见[第十三章·交棒志愿季](13_交棒志愿季.md)。

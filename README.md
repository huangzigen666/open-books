## 建库原则

1. 书只给通用方法框架；个性化诊断 / 陪跑指向问分 / 问路付费服务，不在书内闭合。
2. 政策、批次、分数线必须回官方来源复核。
3. 案例全部脱敏或合成，不含可识别学生信息。
